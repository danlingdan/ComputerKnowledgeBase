# Obsidian 主题与外观自定义

## 一、主题概述

Obsidian 支持自定义主题，你可以通过安装主题或编写 CSS 片段来改变界面外观。

## 二、安装主题

### 1. 从社区主题商店安装

1. 打开 **设置 → Appearance**
2. 在 **Themes** 部分点击 **Browse**
3. 浏览可用主题，点击 **Install** 安装
4. 点击主题名称应用主题

### 2. 手动安装主题

1. 下载主题文件（通常是一个 `.zip` 文件）
2. 解压到 `.obsidian/themes/` 文件夹
3. 在设置中选择主题

### 3. 推荐主题

- **Minimal**：简洁现代，支持自定义颜色
- **Blue Topaz**：蓝色主题，专业感强
- **AnuPpuccin**：粉色主题，可爱风格
- **Catppuccin**：暖色调主题，舒适阅读
- **Obsidianite**：深色主题，适合夜间使用

## 三、调整外观设置

### 1. 字体设置

在 **Settings → Appearance → Font** 中调整：
- **Font**：选择字体
- **Font size**：调整字体大小
- **Line height**：调整行高
- **Letter spacing**：调整字间距

### 2. 颜色设置

在 **Settings → Appearance → Colors** 中调整：
- **Accent color**：强调色
- **Base color scheme**：浅色/深色模式
- **Theme**：选择主题

### 3. 界面设置

在 **Settings → Appearance → Interface** 中调整：
- **Show ribbon**：显示/隐藏侧边栏图标
- **Show status bar**：显示/隐藏状态栏
- **Tab bar mode**：标签栏模式
- **Window frame style**：窗口边框样式

## 四、CSS 片段自定义

### 1. 创建 CSS 片段文件夹

1. 在 Vault 根目录创建 `.obsidian/snippets/` 文件夹
2. 在该文件夹中创建 `.css` 文件

### 2. 启用 CSS 片段

1. 打开 **Settings → Appearance → CSS snippets**
2. 点击 **Reload snippets**
3. 启用要使用的 CSS 片段

### 3. 常用 CSS 片段

#### 修改标题样式

```css
.markdown-preview-view h1 {
    font-size: 2em;
    font-weight: bold;
    color: #2c3e50;
    border-bottom: 2px solid #3498db;
    padding-bottom: 0.2em;
}

.markdown-preview-view h2 {
    font-size: 1.6em;
    font-weight: bold;
    color: #34495e;
}
```

#### 修改链接样式

```css
.markdown-preview-view a {
    color: #3498db;
    text-decoration: none;
    border-bottom: 1px dashed #3498db;
}

.markdown-preview-view a:hover {
    color: #2980b9;
    border-bottom: 1px solid #2980b9;
}
```

#### 修改代码块样式

```css
.markdown-preview-view pre {
    background-color: #2d2d2d;
    border-radius: 8px;
    padding: 1em;
    overflow-x: auto;
}

.markdown-preview-view code {
    background-color: #f8f9fa;
    padding: 0.2em 0.4em;
    border-radius: 4px;
}
```

#### 修改表格样式

```css
.markdown-preview-view table {
    border-collapse: collapse;
    width: 100%;
}

.markdown-preview-view th,
.markdown-preview-view td {
    border: 1px solid #ddd;
    padding: 8px;
    text-align: left;
}

.markdown-preview-view th {
    background-color: #f2f2f2;
}
```

#### 修改列表样式

```css
.markdown-preview-view ul {
    list-style-type: disc;
    padding-left: 1.5em;
}

.markdown-preview-view ol {
    list-style-type: decimal;
    padding-left: 1.5em;
}

.markdown-preview-view li {
    margin-bottom: 0.3em;
}
```

#### 修改引用样式

```css
.markdown-preview-view blockquote {
    border-left: 4px solid #3498db;
    padding-left: 1em;
    margin-left: 0;
    color: #666;
    font-style: italic;
}
```

#### 修改任务列表样式

```css
.markdown-preview-view ul.contains-task-list li.task-list-item {
    list-style: none;
    margin-left: -1.5em;
}

.markdown-preview-view input[type="checkbox"] {
    margin-right: 0.5em;
}

.markdown-preview-view li.task-list-item.is-checked {
    text-decoration: line-through;
    color: #999;
}
```

#### 修改编辑器样式

```css
/* 修改编辑器字体 */
.CodeMirror {
    font-family: 'JetBrains Mono', 'Fira Code', monospace;
    font-size: 14px;
}

/* 修改编辑器背景 */
.theme-dark .CodeMirror {
    background-color: #1e1e1e;
}

.theme-light .CodeMirror {
    background-color: #fafafa;
}
```

### 4. CSS 片段管理

- **组织片段**：将相关的 CSS 片段放在一起
- **命名规范**：使用描述性的文件名
- **备份片段**：使用 Git 备份 CSS 片段

## 五、创建自定义主题

### 1. 主题结构

```
主题文件夹/
├── manifest.json     - 主题配置
├── styles.css        - 主样式文件
├── README.md         - 主题说明
└── screenshots/      - 截图文件夹
```

### 2. manifest.json

```json
{
    "name": "My Theme",
    "version": "1.0.0",
    "description": "My custom theme",
    "author": "Your Name",
    "authorUrl": "https://example.com",
    "minAppVersion": "0.13.0",
    "isDesktopOnly": false
}
```

### 3. 发布主题

1. 创建 GitHub 仓库
2. 上传主题文件
3. 在 Obsidian 论坛发布主题
4. 提交到 Obsidian 社区主题商店

## 六、练习建议

2. 创建一个简单的 CSS 片段修改标题样式
3. 创建一个 CSS 片段修改链接样式
4. 创建一个完整的自定义主题
5. 分享你的 CSS 片段或主题到社区

## 常见问题

**Q：CSS 片段保存在哪里？**

A：CSS 片段保存在 `.obsidian/snippets/` 文件夹中。

**Q：如何禁用 CSS 片段？**

A：在 **Settings → Appearance → CSS snippets** 中取消勾选片段。

**Q：如何创建自定义主题？**

A：创建主题文件夹，编写 `manifest.json` 和 `styles.css`，然后提交到社区主题商店。
