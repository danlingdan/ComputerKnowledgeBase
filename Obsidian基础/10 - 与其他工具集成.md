# Obsidian 与其他工具集成

## 一、集成概述

Obsidian 可以与多种工具集成，扩展功能，提高工作效率。

## 二、与 Git 集成

### 1. 安装 Obsidian Git 插件

1. 打开 **设置 → Community plugins**
2. 搜索 **Obsidian Git** 并安装
3. 启用插件

### 2. 配置 Git

1. 确保本地已安装 Git
2. 在 Vault 根目录初始化 Git 仓库：
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```
3. 添加远程仓库：
   ```bash
   git remote add origin <remote-url>
   git push -u origin main
   ```

### 3. 使用 Obsidian Git

- **自动备份**：设置定时自动提交
- **手动备份**：点击状态栏的 Git 图标
- **查看历史**：使用 Git 命令查看版本历史

### 4. 配置 .gitignore

```gitignore
.obsidian/*
!.obsidian/plugins/
!.obsidian/snippets/
.trash/
*.log
.DS_Store
```

## 三、与浏览器集成

### 1. 使用 Clipper 插件

1. 安装 **Obsidian Clipper** 插件
2. 在浏览器中安装对应扩展
3. 配置 Clipper 设置
4. 在网页中点击 Clipper 按钮保存到 Obsidian

### 2. 使用分享功能

1. 在浏览器中复制网页链接
2. 在 Obsidian 中创建笔记
3. 使用 `[网页标题](网页链接)` 格式添加链接
4. 可以使用插件自动生成笔记

### 3. 推荐浏览器扩展

- **Obsidian Clipper**：官方浏览器扩展
- **MarkDownload**：将网页保存为 Markdown
- **Save to Obsidian**：快速保存到 Obsidian

- 定期检查同步状态

### 2. 同步冲突

## 常见问题

**Q：如何在多设备间同步笔记？**

A：可以使用 Obsidian Sync、Git、云存储（iCloud、OneDrive、Dropbox）等方式。

**Q：如何导入其他笔记工具的内容？**

A：通常可以导出为 Markdown 格式，然后导入到 Obsidian。

**Q：集成会影响 Obsidian 性能吗？**

A：过多的集成可能会影响性能，建议只启用必要的集成。
