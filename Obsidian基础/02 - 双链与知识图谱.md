# Obsidian 双链与知识图谱

## 一、什么是双链

### 概念

双链（双向链接）是 Obsidian 的核心功能，它允许笔记之间相互引用，形成知识网络。

与传统的超链接不同，双链不仅记录了从 A 到 B 的链接，同时也记录了从 B 到 A 的反向链接。

### 为什么使用双链

- **构建知识网络**：笔记不再是孤立的文档，而是相互关联的知识节点
- **发现隐藏联系**：通过反向链接发现相关的笔记和想法
- **强化记忆**：通过关联学习，加深对知识的理解和记忆

## 二、创建双链

### 基本语法

```markdown
[[笔记名称]]
```

### 创建方式

1. **手动输入**：直接输入 `[[`，Obsidian 会自动提示匹配的笔记名称
2. **快捷键**：选中文本后按 `Ctrl/Cmd + K`
3. **拖拽**：从文件面板拖拽笔记到当前编辑位置

### 示例

```markdown
我正在学习 [[Markdown基础]]，它是 [[Obsidian]] 的基础语法。
```

## 三、链接类型

### 1. 链接到现有笔记

```markdown
[[已存在的笔记]]
```

### 2. 链接到不存在的笔记

```markdown
[[将要创建的笔记]]
```

点击蓝色链接即可创建新笔记。

### 3. 链接到笔记中的标题

```markdown
[[笔记名称#标题名称]]
[[Markdown基础#一、标题]]
```

### 4. 链接到笔记中的块

```markdown
[[笔记名称^块引用]]
```

### 5. 链接到标签

```markdown
#标签名称
#学习 #编程
```

### 6. 别名链接

```markdown
[[笔记名称|显示文本]]
[[Markdown基础|MD基础]]
```

## 四、反向链接

### 概念

反向链接显示所有引用了当前笔记的其他笔记。

### 查看方式

- 在笔记右侧的 **Backlinks** 面板中查看
- 打开笔记时，底部会显示反向链接列表

### 示例场景

```markdown
笔记 A 中写了：[[笔记 B]]
笔记 C 中写了：[[笔记 B]]

在笔记 B 的反向链接中会显示：笔记 A、笔记 C
```

## 五、知识图谱

### 什么是知识图谱

知识图谱是 Obsidian 中可视化展示笔记之间链接关系的功能。

### 打开图谱

- 点击左侧边栏的 **Graph View** 图标
- 使用快捷键 `Ctrl/Cmd + G`

### 图谱视图类型

1. **全局图谱**：显示所有笔记及其链接
2. **局部图谱**：以当前笔记为中心，显示相关笔记
3. **搜索图谱**：显示与搜索关键词相关的笔记

### 图谱自定义

- **颜色**：根据标签或文件夹为节点着色
- **大小**：根据笔记的链接数量调整节点大小
- **过滤**：按标签、文件夹、链接数等条件过滤
- **布局**：选择不同的布局算法（力导向、同心圆等）

### 图谱中的交互

- **点击节点**：打开对应的笔记
- **拖拽节点**：调整节点位置
- **滚轮缩放**：放大/缩小图谱
- **右键菜单**：查看节点信息、复制链接等

## 六、标签

### 创建标签

```markdown
#标签名称
#学习 #Python #数据分析
```

### 嵌套标签

```markdown
#编程/Python
#编程/JavaScript
#学习/数学
```

### 使用标签

- 在搜索栏中搜索标签：`#标签名称`
- 在标签面板中按标签浏览笔记
- 在图谱中按标签筛选节点

## 七、双链最佳实践

### 1. 保持笔记原子化

每个笔记只讨论一个主题，便于链接和复用。

### 2. 使用描述性标题

```markdown
# 好标题：[[Python函数定义]]
# 坏标题：[[笔记1]]
```

### 3. 建立层次结构

```markdown
[[编程语言]]
  ├── [[Python]]
  ├── [[JavaScript]]
  └── [[Java]]
```

### 4. 使用标签分类

```markdown
#编程语言/Python
#学习/入门
```

### 5. 定期回顾反向链接

查看哪些笔记引用了当前笔记，发现潜在的关联。

### 6. 使用图谱发现模式

通过图谱视图发现知识网络中的密集区域和孤立节点。

## 八、高级技巧

### 1. 链接到附件

```markdown
[[附件/image.png]]
[[附件/document.pdf]]
```

### 2. 使用模板创建链接

在模板中预定义常用的链接结构。

### 3. 批量创建链接

使用搜索替换功能批量添加链接。

### 4. 利用插件增强功能

- **Link Explorer**：可视化链接关系
- **Tag Wrangler**：管理和重命名标签
- **Natural Language Dates**：自动识别日期并创建链接

## 九、练习建议

1. 创建 5-10 个主题相关的笔记，并在它们之间建立双链
2. 在图谱视图中查看这些笔记的链接关系
3. 创建一些嵌套标签，并为笔记添加标签
4. 查看某个笔记的反向链接，了解它被哪些笔记引用
5. 尝试链接到笔记中的特定标题和块

## 常见问题

**Q：双链与普通链接有什么区别？**

A：双链会自动建立反向链接，便于追踪笔记之间的引用关系。普通链接只能单向访问。

**Q：如何删除双链？**

A：直接删除 `[[笔记名称]]` 即可，Obsidian 会自动更新反向链接。

**Q：图谱视图太乱怎么办？**

A：可以使用过滤功能按标签或文件夹筛选，或者调整布局和缩放比例。
