# Obsidian 高级搜索技巧

## 一、搜索概述

Obsidian 的搜索功能非常强大，可以帮助你快速定位笔记内容。了解高级搜索技巧可以大大提高工作效率。

## 二、基本搜索

### 1. 打开搜索面板

- **快捷键**：`Ctrl/Cmd + Shift + F`
- **命令面板**：`Ctrl/Cmd + P`，输入 `Search`

### 2. 简单搜索

```markdown
搜索关键词：python
```

### 3. 搜索结果

搜索结果会显示：
- 匹配的笔记列表
- 匹配位置的预览
- 匹配数量统计

## 三、高级搜索语法

### 1. 精确匹配

```markdown
"搜索短语"
"python 函数定义"
```

### 2. 排除关键词

```markdown
搜索关键词 -排除词
python -java
```

### 3. 或搜索

```markdown
关键词1 OR 关键词2
python OR java
```

### 4. 模糊搜索

```markdown
~关键词
~pytho
```

### 5. 正则表达式

```markdown
/regex/
/^[A-Z]/
```

## 四、搜索修饰符

### 1. 按标签搜索

```markdown
tag:#标签名
tag:#编程
tag:#编程/Python
```

### 2. 按链接搜索

```markdown
link:[[笔记名称]]
link:[[Python]]
```

### 3. 按文件类型搜索

```markdown
file:md
file:pdf
file:image
```

### 4. 按路径搜索

```markdown
path:文件夹名
path:编程
path:编程/Python
```

### 5. 按标题搜索

```markdown
title:关键词
title:Python
```

### 6. 按内容搜索

```markdown
content:关键词
content:函数定义
```

### 7. 组合搜索

```markdown
tag:#编程 file:md
path:编程 tag:#Python
title:笔记 content:教程
```

## 五、搜索结果过滤与排序

### 1. 过滤结果

- **文件类型**：点击搜索结果上方的文件类型图标
- **标签**：点击搜索结果上方的标签
- **路径**：点击搜索结果上方的路径

### 2. 排序结果

- **相关性**：按匹配程度排序
- **修改时间**：按最后修改时间排序
- **创建时间**：按创建时间排序
- **文件名**：按文件名排序

### 3. 搜索范围

- **全局搜索**：搜索整个 Vault
- **当前文件夹**：只搜索当前文件夹
- **当前笔记**：只搜索当前笔记

## 六、搜索命令与快捷键

### 1. 搜索命令

- **Search in file**：在当前文件中搜索
- **Search in selection**：在选中区域中搜索
- **Search in folder**：在指定文件夹中搜索
- **Search for tags**：搜索标签
- **Search for links**：搜索链接

### 2. 快捷键

- `Ctrl/Cmd + Shift + F`：全局搜索
- `Ctrl/Cmd + F`：在当前文件中搜索
- `Ctrl/Cmd + G`：查找下一个匹配
- `Ctrl/Cmd + Shift + G`：查找上一个匹配

## 七、搜索插件推荐

### 1. Search on Steroids

- **功能**：增强搜索功能
- **特点**：支持更多搜索修饰符、搜索历史、高级过滤

### 2. Dataview

- **功能**：基于数据的查询
- **特点**：支持复杂查询、自定义视图、数据表格

### 3. Quick Switcher++

- **功能**：增强快速切换
- **特点**：支持模糊搜索、标签搜索、路径搜索

## 八、搜索技巧

### 1. 使用通配符

```markdown
py*          - 匹配以 py 开头的词
*thon        - 匹配以 thon 结尾的词
p*n          - 匹配 p 开头、n 结尾的词
```

### 2. 使用分组

```markdown
(python OR java) AND 教程
```

### 3. 搜索特定字符

```markdown
\#tag        - 搜索 #tag 文本（而非标签）
\[link\]     - 搜索 [link] 文本（而非链接）
```

### 4. 保存搜索

1. 执行搜索
2. 点击搜索面板右上角的 **Save** 按钮
3. 命名搜索并保存

### 5. 使用搜索模板

```markdown
tag:#待办 AND -[x]        - 搜索未完成的待办事项
tag:#阅读 AND !completed  - 搜索未完成的阅读笔记
```

## 九、练习建议

1. 使用高级搜索语法搜索笔记
2. 创建保存的搜索
3. 使用 Dataview 创建自定义查询
4. 安装 Search on Steroids 插件
5. 练习组合搜索多个条件

## 常见问题

**Q：如何搜索标签？**

A：使用 `tag:#标签名` 语法，如 `tag:#编程`。

**Q：如何搜索链接？**

A：使用 `link:[[笔记名称]]` 语法，如 `link:[[Python]]`。

**Q：如何保存搜索结果？**

A：点击搜索面板右上角的 **Save** 按钮，命名并保存搜索。
