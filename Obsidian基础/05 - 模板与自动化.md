# Obsidian 模板与自动化

## 一、模板概述

模板是预先定义好的笔记格式，使用模板可以快速创建结构一致的笔记，提高写作效率。

### 模板类型

- **内置模板**：Obsidian 自带的简单模板系统
- **Templater 插件**：支持 JavaScript 的高级模板系统

## 二、内置模板系统

### 1. 创建模板文件夹

1. 在 Vault 根目录创建 `.obsidian/templates/` 文件夹
2. 在该文件夹中创建模板文件

### 2. 创建模板文件

```markdown
---
标题：
日期：
标签：
---

## 内容

## 参考链接
```

### 3. 使用模板

1. 按 `Ctrl/Cmd + P` 打开命令面板
2. 输入 `Insert Template`
3. 选择要使用的模板

## 三、Templater 插件

### 1. 安装 Templater

1. 打开 **设置 → Community plugins**
2. 搜索 **Templater** 并安装
3. 启用插件

### 2. 配置 Templater

1. 打开 Templater 设置
2. 设置模板文件夹路径（建议 `.obsidian/templates/`）
3. 启用 **Trigger on new file creation**

### 3. 模板语法

#### 基础变量

```markdown
<% tp.file.title %>          - 当前文件名
<% tp.file.name %>           - 当前文件名（不含扩展名）
<% tp.file.path %>           - 当前文件路径
<% tp.file.folder %>         - 当前文件所在文件夹
<% tp.date.now() %>          - 当前日期时间
<% tp.date.yesterday() %>    - 昨天日期
<% tp.date.tomorrow() %>     - 明天日期
```

#### 日期格式

```markdown
<% tp.date.now("YYYY-MM-DD") %>           - 2024-01-15
<% tp.date.now("YYYY-MM-DD HH:mm") %>     - 2024-01-15 10:30
<% tp.date.now("dddd, MMMM Do YYYY") %>   - Monday, January 15th 2024
```

#### 用户输入

```markdown
<% tp.prompt("请输入标题") %>              - 输入文本
<% tp.system.suggester(["选项1", "选项2"], ["值1", "值2"]) %>  - 选择选项
```

#### 文件操作

```markdown
<% tp.file.rename("新文件名") %>           - 重命名文件
<% tp.file.move("新路径") %>               - 移动文件
<% tp.file.create_new(tp.file.find_template("模板名"), "新文件名") %>  - 创建新文件
```

#### 代码块

```markdown
<%* 
let title = tp.prompt("请输入标题");
let tags = tp.prompt("请输入标签");
-%>
---
标题：<% title %>
标签：#<% tags %>
---
```

## 五、自动化工作流

### 1. 每日笔记自动创建

1. 安装 **Daily Notes** 核心插件或 **Calendar** 社区插件
2. 配置每日笔记模板
3. 设置自动创建时间

### 2. 使用 QuickAdd 快速创建

1. 安装 **QuickAdd** 插件
2. 创建宏命令
3. 绑定快捷键

### 3. 使用 Templater 自动化

```markdown
<%*
// 自动创建相关笔记
await tp.file.create_new(tp.file.find_template("会议记录"), `会议记录-${tp.date.now("YYYY-MM-DD")}`);
-%>
```

## 常见问题

**Q：模板文件保存在哪里？**

A：模板文件默认保存在 `.obsidian/templates/` 文件夹中。

**Q：如何在模板中使用 JavaScript？**

A：使用 Templater 插件，在 `<%* ... %>` 块中编写 JavaScript 代码。

**Q：如何让模板自动应用？**

A：在 Templater 设置中启用 **Trigger on new file creation**，并设置默认模板。
