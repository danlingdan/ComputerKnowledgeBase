# Obsidian 发布与分享

## 一、发布概述

Obsidian 提供多种方式发布和分享笔记，包括导出为不同格式、使用 Obsidian Publish 服务等。

## 二、导出笔记

### 1. 导出为 PDF

1. 打开要导出的笔记
2. 点击右上角的 **更多选项**（三个点）
3. 选择 **Export to PDF**
4. 设置导出选项（页面大小、边距、主题等）
5. 点击 **Export** 保存

### 2. 导出为 HTML

1. 打开要导出的笔记
2. 点击右上角的 **更多选项**（三个点）
3. 选择 **Export to HTML**
4. 设置导出选项（包含附件、嵌入笔记等）
5. 点击 **Export** 保存

### 3. 导出为 Markdown

1. 打开要导出的笔记
2. 点击右上角的 **更多选项**（三个点）
3. 选择 **Copy as Markdown** 或直接复制内容
4. 粘贴到其他编辑器中

### 4. 批量导出

1. 安装 **Export All** 插件
2. 配置导出选项
3. 选择要导出的笔记
4. 批量导出

## 三、Obsidian Publish

### 1. 什么是 Obsidian Publish

Obsidian Publish 是官方提供的发布服务，可以将你的笔记发布到云端，生成一个可访问的网站。

### 2. 订阅服务

1. 打开 **设置 → Publish**
2. 点击 **Subscribe** 订阅服务
3. 选择订阅计划

### 3. 发布笔记

1. 在笔记的 YAML 前端添加 `publish: true`
2. 打开 **设置 → Publish**
3. 选择要发布的笔记
4. 点击 **Publish**

### 4. 自定义发布站点

1. 打开 **设置 → Publish**
2. 点击 **Site options**
3. 设置站点标题、描述、主题等
4. 配置导航菜单和首页

### 5. 发布选项

- **Frontmatter**：是否显示 YAML 前端
- **Backlinks**：是否显示反向链接
- **Graph**：是否显示图谱
- **Search**：是否启用搜索功能
- **Tags**：是否显示标签

## 四、分享笔记

### 1. 分享单个笔记

1. 打开要分享的笔记
2. 点击右上角的 **更多选项**（三个点）
3. 选择 **Copy link to note**
4. 将链接发送给他人

### 2. 分享多个笔记

1. 选择多个笔记
2. 右键点击，选择 **Copy links**
3. 将链接发送给他人

### 3. 分享图谱视图

1. 打开图谱视图
2. 调整视图设置
3. 截图或录屏分享

### 4. 分享到社交媒体

1. 复制笔记链接
2. 粘贴到社交媒体平台
3. 添加描述和标签

## 五、与他人协作

### 1. 使用 Obsidian Sync

1. 订阅 Obsidian Sync 服务
2. 配置同步设置
3. 邀请协作者
4. 实时同步笔记

### 2. 使用 Git 协作

1. 将 Vault 推送到 Git 仓库
2. 邀请协作者加入仓库
3. 定期拉取和推送更新
4. 解决合并冲突

### 3. 使用第三方工具

- **Notion**：导出笔记到 Notion
- **Confluence**：导出笔记到 Confluence
- **Slack**：分享笔记到 Slack

## 六、发布注意事项

### 1. 隐私保护

- 发布前检查笔记内容是否包含敏感信息
- 使用 Obsidian Publish 的权限控制功能
- 设置密码保护或限制访问

### 2. 内容格式

- 确保导出的内容格式正确
- 检查图片和附件是否正确嵌入
- 测试链接是否可访问

### 3. 性能优化

- 压缩图片大小
- 减少页面加载时间
- 使用缓存优化

## 七、练习建议

1. 导出一篇笔记为 PDF
2. 导出一篇笔记为 HTML
3. 尝试使用 Obsidian Publish（如果有订阅）
4. 分享笔记链接给朋友
5. 配置 Git 协作流程

## 常见问题

**Q：Obsidian Publish 需要付费吗？**

A：是的，Obsidian Publish 是付费服务，提供不同的订阅计划。

**Q：如何设置发布权限？**

A：在 Obsidian Publish 设置中配置访问权限，可以设置密码保护或限制访问。

**Q：导出的 PDF 样式不好看怎么办？**

A：可以尝试更换主题，或使用 CSS 片段自定义导出样式。
