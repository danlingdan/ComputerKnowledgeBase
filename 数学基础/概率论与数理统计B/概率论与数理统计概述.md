# 概率论与数理统计概述

概率论与数理统计是研究随机现象规律性的数学学科。概率论从理论上研究随机现象的规律，数理统计从数据出发研究随机现象的推断和决策。这两门学科为数据分析、机器学习、质量控制、风险评估等领域提供了理论基础。

## 概率论基础

### 随机事件与概率

**随机试验**：具有以下特征的试验：
1. 可以在相同条件下重复进行
2. 每次试验的结果不止一个
3. 试验前无法确定具体结果

**样本空间**：随机试验所有可能结果的集合，记为 $\Omega$。

**随机事件**：样本空间的子集，记为 $A \subseteq \Omega$。

**概率定义**：
- **古典概型**：$P(A) = \frac{A包含的样本点数}{\Omega中样本点总数}$
- **几何概型**：$P(A) = \frac{A的度量}{\Omega的度量}$
- **统计定义**：频率的极限值
- **公理化定义**：满足非负性、规范性、可列可加性的函数

### 条件概率与独立性

**条件概率**：在事件 $B$ 发生的条件下事件 $A$ 发生的概率：

$$P(A|B) = \frac{P(AB)}{P(B)}, \quad P(B) > 0$$

**乘法公式**：
$$P(AB) = P(A)P(B|A) = P(B)P(A|B)$$

**全概率公式**：设 $B_1, B_2, \ldots, B_n$ 为完备事件组，则：

$$P(A) = \sum_{i=1}^{n} P(B_i)P(A|B_i)$$

**贝叶斯公式**：

$$P(B_i|A) = \frac{P(B_i)P(A|B_i)}{\sum_{j=1}^{n} P(B_j)P(A|B_j)}$$

**独立性**：事件 $A$ 与 $B$ 独立当且仅当 $P(AB) = P(A)P(B)$。

### 随机变量及其分布

**随机变量**：定义在样本空间上的实值函数，记为 $X(\omega)$。

**分布函数**：

$$F(x) = P(X \leq x), \quad -\infty < x < +\infty$$

分布函数的性质：
1. 单调不减：$x_1 < x_2$ 时 $F(x_1) \leq F(x_2)$
2. 右连续：$\lim_{x \to x_0^+} F(x) = F(x_0)$
3. 规范性：$F(-\infty) = 0$，$F(+\infty) = 1$

**离散型随机变量**：取有限或可列个值的随机变量。

- **概率分布**：$P(X = x_i) = p_i, \quad i = 1, 2, \ldots$
- **分布律**：$p_i \geq 0$，$\sum_{i} p_i = 1$

**连续型随机变量**：存在非负函数 $f(x)$ 使得：

$$F(x) = \int_{-\infty}^{x} f(t)dt$$

- **概率密度函数**：$f(x) \geq 0$，$\int_{-\infty}^{+\infty} f(x)dx = 1$

### 常见分布

**离散型分布**：
- **0-1分布**：$P(X=1)=p, P(X=0)=1-p$
- **二项分布**：$X \sim B(n,p)$，$P(X=k) = C_n^k p^k(1-p)^{n-k}$
- **泊松分布**：$X \sim P(\lambda)$，$P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$
- **几何分布**：$P(X=k) = (1-p)^{k-1}p$

**连续型分布**：
- **均匀分布**：$X \sim U(a,b)$，$f(x) = \frac{1}{b-a}, a \leq x \leq b$
- **指数分布**：$X \sim E(\lambda)$，$f(x) = \lambda e^{-\lambda x}, x \geq 0$
- **正态分布**：$X \sim N(\mu,\sigma^2)$，$f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$

### 多维随机变量

**联合分布函数**：

$$F(x,y) = P(X \leq x, Y \leq y)$$

**边缘分布**：
- $F_X(x) = F(x, +\infty)$
- $F_Y(y) = F(+\infty, y)$

**条件分布**：
- 离散型：$P(X=x_i|Y=y_j) = \frac{P(X=x_i,Y=y_j)}{P(Y=y_j)}$
- 连续型：$f_{X|Y}(x|y) = \frac{f(x,y)}{f_Y(y)}$

**独立性**：$X$ 与 $Y$ 独立当且仅当 $F(x,y) = F_X(x)F_Y(y)$。

### 随机变量的数字特征

**数学期望**（均值）：
- 离散型：$E(X) = \sum_{i} x_i p_i$
- 连续型：$E(X) = \int_{-\infty}^{+\infty} x f(x)dx$

**方差**：
$$D(X) = E[X - E(X)]^2 = E(X^2) - [E(X)]^2$$

**标准差**：$\sigma(X) = \sqrt{D(X)}$

**协方差**：
$$\text{Cov}(X,Y) = E[(X-E(X))(Y-E(Y))] = E(XY) - E(X)E(Y)$$

**相关系数**：
$$\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sqrt{D(X)}\sqrt{D(Y)}}$$

**期望和方差的性质**：
- $E(aX + b) = aE(X) + b$
- $D(aX + b) = a^2D(X)$
- $E(X + Y) = E(X) + E(Y)$
- 若 $X,Y$ 独立，则 $D(X + Y) = D(X) + D(Y)$

### 大数定律与中心极限定理

**大数定律**：
- **切比雪夫大数定律**：独立随机变量序列的算术平均依概率收敛于期望
- **伯努利大数定律**：频率依概率收敛于概率
- **辛钦大数定律**：独立同分布随机变量序列的算术平均依概率收敛于期望

**中心极限定理**：
- **林德伯格-列维定理**：独立同分布随机变量之和的标准化分布收敛于标准正态分布
- **棣莫弗-拉普拉斯定理**：二项分布的标准化收敛于标准正态分布

## 数理统计基础

### 总体与样本

**总体**：研究对象的全体，用随机变量 $X$ 表示。

**样本**：从总体中抽取的 $n$ 个个体 $X_1, X_2, \ldots, X_n$。

**简单随机样本**：满足独立性和同分布性的样本。

**样本容量**：样本中个体的数量 $n$。

### 统计量

**统计量**：样本的函数，不包含未知参数。

**常用统计量**：
- **样本均值**：$\bar{X} = \frac{1}{n}\sum_{i=1}^{n} X_i$
- **样本方差**：$S^2 = \frac{1}{n-1}\sum_{i=1}^{n}(X_i - \bar{X})^2$
- **样本标准差**：$S = \sqrt{S^2}$
- **样本 $k$ 阶矩**：$A_k = \frac{1}{n}\sum_{i=1}^{n} X_i^k$

### 抽样分布

**$\chi^2$分布**：设 $X_1, X_2, \ldots, X_n$ 独立同分布于 $N(0,1)$，则：

$$\chi^2 = \sum_{i=1}^{n} X_i^2 \sim \chi^2(n)$$

性质：$E(\chi^2) = n$，$D(\chi^2) = 2n$

**$t$分布**：设 $X \sim N(0,1)$，$Y \sim \chi^2(n)$，且 $X,Y$ 独立，则：

$$t = \frac{X}{\sqrt{Y/n}} \sim t(n)$$

**$F$分布**：设 $X \sim \chi^2(n_1)$，$Y \sim \chi^2(n_2)$，且 $X,Y$ 独立，则：

$$F = \frac{X/n_1}{Y/n_2} \sim F(n_1, n_2)$$

### 参数估计

**点估计**：用样本统计量估计总体参数。

**估计方法**：
- **矩估计**：用样本矩估计总体矩
- **最大似然估计**：使似然函数最大的参数值

**估计量的评价标准**：
- **无偏性**：$E(\hat{\theta}) = \theta$
- **有效性**：方差最小的无偏估计
- **一致性**：样本量增大时估计量收敛于真实值

**区间估计**：给出参数可能落入的区间。

**置信区间**：$(\hat{\theta}_L, \hat{\theta}_U)$ 满足 $P(\hat{\theta}_L < \theta < \hat{\theta}_U) = 1-\alpha$

### 假设检验

**基本概念**：
- **原假设** $H_0$：待检验的假设
- **备择假设** $H_1$：与原假设对立的假设
- **显著性水平** $\alpha$：犯第一类错误的概率
- **拒绝域**：拒绝原假设的样本值区域

**两类错误**：
- **第一类错误**：$H_0$ 为真时拒绝 $H_0$（显著性水平 $\alpha$）
- **第二类错误**：$H_0$ 为假时接受 $H_0$（概率 $\beta$）

**检验步骤**：
1. 建立原假设和备择假设
2. 选择适当的检验统计量
3. 确定拒绝域
4. 计算检验统计量的值
5. 做出统计决策

**常用检验**：
- **$Z$检验**：正态总体均值检验（方差已知）
- **$t$检验**：正态总体均值检验（方差未知）
- **$\chi^2$检验**：正态总体方差检验
- **$F$检验**：两个正态总体方差比检验

### 回归分析

**一元线性回归**：
$$Y = \beta_0 + \beta_1 x + \varepsilon$$

其中 $\varepsilon \sim N(0,\sigma^2)$。

**最小二乘估计**：
$$\hat{\beta}_1 = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(Y_i - \bar{Y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2}$$
$$\hat{\beta}_0 = \bar{Y} - \hat{\beta}_1\bar{x}$$

**决定系数**：$R^2 = \frac{\text{回归平方和}}{\text{总平方和}}$
