# 数学基础局部规则

- 明确定义对象、符号、定义域、参数范围和前置条件；不要只记录公式结论。
- 区分定义、公理、定理、性质、推论、证明、推导、例题和应用。引用定理时同时说明成立条件。
- 推导保留关键等价变形与依据；跳步可能影响理解时补充中间步骤，不用“显然”掩盖关键论证。
- 题目笔记区分“原题、思路、解答、检验、易错点”。保留原题来源；不要用整理后的题解覆盖唯一原始记录。
- 概率统计笔记明确随机试验、样本空间、随机变量类型、分布及参数、独立性假设、统计量和结论含义。
- 数值结果注明精确值或近似值、误差和舍入规则；数值分析内容补充收敛条件、误差阶与稳定性。
- 复习讲义和公式清单作为导航与复习材料，不替代概念笔记；链接到详细定义、证明和典型题。
- 离散数学若保留在 `编程与算法`，通过链接连接相关数学概念；未经确认不要移动课程目录。
