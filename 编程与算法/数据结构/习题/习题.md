**4.** 存储结构由哪两种基本的存储方法实现？

### 时间复杂度分析

分析下列各段算法的时间复杂度。

**（1）**

```c
x = 90;
y = 100;
while (y > 0)
    if (x > 100) {
        x = x - 10;
        y--;
    } else {
        x++;
    }
```

**（2）**

```c
for (i = 0; i < n; i++)
    for (j = 0; j < m; j++)
        a[i][j] = 0;
```

**（3）**

```c
s = 0;
for (i = 0; i < n; i++)
    for (j = 0; j < n; j++)
```

```c
i = 1;

    i = i * 3;
```

**（5）**

```c
x = 0;
for (i = 1; i < n; i++)
    for (j = 1; j <= n - i; j++)
```
