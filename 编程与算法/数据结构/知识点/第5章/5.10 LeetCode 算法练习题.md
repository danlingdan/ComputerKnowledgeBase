> [!example] 算法练习题 5.1 LeetCode 872 叶子相似的树 ★
> **【问题描述】** 给定根分别为 `root1`、`root2` 的两棵树，判断叶相似（叶值序列相同）则返回 `true`。图 5.36 的叶值序列为 $(6,7,4,9,8)$。
> **【问题分析】** 先序遍历两树得叶值序列，比较长度与逐位值。
> **【算法描述】**
> ```cpp
> void PreOrderTraverse(struct TreeNode* node, int* seq, int* seqSize)
> { // 先序遍历方法得到叶值序列
>     if (!node->left && !node->right)
>         seq[(*seqSize)++] = node->val;   // 叶子结点的值添加到序列中
>     else
>     {
>         if (node->left)
>             PreOrderTraverse(node->left, seq, seqSize);
>         if (node->right)
>             PreOrderTraverse(node->right, seq, seqSize);
>     }
> }
> 
> bool leafSimilar(struct TreeNode* root1, struct TreeNode* root2)
> { // 判断两棵二叉树是否叶相似
>     int seq1[200], seq1Size = 0;
>     if (root1)
>         PreOrderTraverse(root1, seq1, &seq1Size);
> 
>     int seq2[200], seq2Size = 0;
>     if (root2)
>         PreOrderTraverse(root2, seq2, &seq2Size);
> 
>     if (seq1Size != seq2Size)
>         return false;
> 
>     for (int i = 0; i < seq1Size; i++)
>     {
>         if (seq1[i] != seq2[i])
>             return false;
>     }
>     return true;
> }
> ```
> **【算法分析】** 遍历两树，时间复杂度 $O(\max(n_1, n_2))$，空间复杂度 $O(\max(n_1, n_2))$。

> [!example] 算法练习题 5.2 LeetCode 113 路径总和 Ⅱ ★★
> **【问题描述】** 给定根 `root` 与目标和 `targetSum`，找出所有根到叶子且路径和等于目标和的路径。输入 `root=[5,4,8,11,NULL,13,4,7,2,NULL,NULL,5,1]`、`targetSum=22`，输出 `[[5,4,11,2],[5,8,4,5]]`（图 5.38）。
> **【问题分析】** 先序遍历记录根到当前结点的路径和，枚举每条根到叶子路径，叶子且路径和等于目标和则记录。
> **【算法描述】**
> ```cpp
> int** ret; int retSize; int* retColSize; int* path; int pathSize;
> 
> void PreOrderTraverse(struct TreeNode* root, int targetSum)
> { // 先序遍历记录每条根到叶子路径
>     if (root == NULL) return;
>     path[pathSize++] = root->val;
>     targetSum -= root->val;
>     if (root->left == NULL && root->right == NULL && targetSum == 0)
>     {                              // 路径总和等于目标和
>         int* tmp = (int*)malloc(pathSize * sizeof(int));
>         memcpy(tmp, path, sizeof(int) * pathSize);
>         ret[retSize] = tmp;
>         retColSize[retSize++] = pathSize;
>     }
>     PreOrderTraverse(root->left, targetSum);
>     PreOrderTraverse(root->right, targetSum);
>     pathSize--;
> }
> 
> int** pathSum(struct TreeNode* root, int targetSum, int* returnSize, int** returnColumnSizes)
> {
>     ret = (int**)malloc(1000 * sizeof(int*));
>     retColSize = (int*)malloc(1000 * sizeof(int));
>     path = (int*)malloc(1000 * sizeof(int));
>     retSize = pathSize = 0;
>     PreOrderTraverse(root, targetSum);
>     *returnSize = retSize;
>     *returnColumnSizes = retColSize;
>     return ret;
> }
> ```
> **【算法分析】** 最坏情况下路径数与每路径结点数均为 $O(n)$，时间复杂度 $O(n^2)$，空间复杂度 $O(n)$。

> [!example] 算法练习题 5.3 LeetCode 515 在每个树行中找最大值 ★★
> **【问题描述】** 给定根 `root`，找出每层的最大值。输入 `root=[1,3,2,5,3,NULL,9]`，输出 `[1,3,9]`（图 5.39）。
> **【问题分析】** 先序遍历，`curHeight` 标记当前高度，首次到该层直接写入结果，否则更新最大值。
> **【算法描述】**
> ```cpp
> #define MAX_NODE_SIZE 10001
> #define MAX(a, b) ((a) > (b) ? (a) : (b))
> 
> void PreOrderTraverse(int* res, int* pos, struct TreeNode* root, int curHeight)
> { // 先序遍历进行深度优先搜索
>     if (curHeight == *pos)                  // 该层第一个结点直接写入 res
>         res[(*pos)++] = root->val;
>     else
>         res[curHeight] = MAX(res[curHeight], root->val); // 更新该层最大值
>     if (root->left)
>         PreOrderTraverse(res, pos, root->left, curHeight + 1);
>     if (root->right)
>         PreOrderTraverse(res, pos, root->right, curHeight + 1);
> }
> 
> int* largestValues(struct TreeNode* root, int* returnSize)
> { // 寻找每层的最大值
>     if (!root)
>     {
>         *returnSize = 0;
>         return NULL;
>     }
>     int* res = (int*)malloc(MAX_NODE_SIZE * sizeof(int));
>     *returnSize = 0;
>     PreOrderTraverse(res, returnSize, root, 0);
>     return res;
> }
> ```
> **【算法分析】** 每结点访问一次，时间复杂度 $O(n)$，空间复杂度 $O(n)$。
