# 5.9 案例分析与实现

> [!nav] 导航
> 上一知识点：[[5.08.02 并查集的实现及优化]] · [[MOC - 第5章 树和二叉树|本章目录]] · [[MOC - 数据结构|课程总览]] · 下一知识点：[[5.10 LeetCode 算法练习题]]

案例 5.1 已在 5.7 节讨论；案例 5.2 提出用二叉树表示表达式，本节给出表达式树的构建与求值算法。

> [!example] 案例 5.2：利用二叉树求解表达式的值
> **【案例分析】** 任意算术表达式可用二叉树表示（叶子为操作数、分支为双目运算符）。创建表达式树需按运算符优先级比较，类似第 3 章表达式求值的运算符栈，另用**表达式树栈 EXPT** 暂存已建好的树根，使其作为另一运算符结点的子树。
> **【案例实现】** 用两个栈：OPTR 暂存运算符、EXPT 暂存表达式树根。假设表达式以 “#” 开始、以 “#” 结束。

> [!example] 算法 5.16 表达式树的创建
> **【算法步骤】** ① 初始化 OPTR、EXPT，将 “#” 压入 OPTR；② 读 ch，未遇 “#” 或 OPTR 栈顶非 “#” 时循环：ch 非运算符则建单结点树压 EXPT；ch 是运算符则按 OPTR 栈顶与 ch 优先级比较——`<` 压 ch，`>` 弹运算符与两子树建新树压 EXPT，`=` 弹 “(”（括号匹配）。
> **【算法描述】**
> ```cpp
> void InitExpTree()
> { // 表达式树的创建算法
>     InitStack(EXPT);                  // 初始化 EXPT 栈
>     InitStack(OPTR);                  // 初始化 OPTR 栈
>     Push(OPTR, '#');                  // 将表达式起始符 "#" 压入 OPTR 栈
>     cin >> ch;
>     while (ch != '#' || GetTop(OPTR) != '#')  // 表达式未读完或 OPTR 栈顶非 "#"
>     {
>         if (!In(ch))                  // ch 不是运算符
>         {
>             CreateExpTree(T, NULL, NULL, ch);  // 以 ch 为根创建单结点二叉树
>             Push(EXPT, T);            // 将二叉树根结点 T 进 EXPT 栈
>             cin >> ch;                // 读入下一字符
>         }
>         else
>             switch (Precede(GetTop(OPTR), ch)) // 比较 OPTR 栈顶与 ch 的优先级
>             {
>                 case '<':
>                     Push(OPTR, ch); cin >> ch; // 当前字符 ch 压入 OPTR 栈
>                     break;
>                 case '>':
>                     Pop(OPTR, theta);          // 弹出 OPTR 栈顶运算符
>                     Pop(EXPT, b); Pop(EXPT, a); // 弹出 EXPT 栈顶两个操作数
>                     CreateExpTree(T, a, b, theta); // 以 theta 为根，a 左、b 右建树
>                     Push(EXPT, T);             // 树根 T 进 EXPT 栈
>                     break;
>                 case '=':
>                     Pop(OPTR, x); cin >> ch;   // 弹出 OPTR 栈顶的 "("
>                     break;
>             } // switch
>     } // while
> }
> ```
> **【算法分析】** 读取每个字符一次，长度 $n$ 则时间复杂度 $O(n)$；辅助空间为 OPTR 与 EXPT 栈，空间复杂度亦 $O(n)$。

> [!example] 算法 5.17 表达式树的求值
> **【算法步骤】** ① `lvalue`、`rvalue` 初为 0；② 当前结点为叶子则返回其数值，否则递归算左、右子树值，按运算符类型运算返回。
> **【算法描述】**
> ```cpp
> int EvaluateExpTree(BiTree T)
> { // 遍历表达式树进行表达式求值
>     lvalue = rvalue = 0;               // 初始为 0
>     if (T->lchild == NULL && T->rchild == NULL)
>         return T->data - '0';          // 结点为操作数，返回其数值
>     else                               // 结点为运算符
>     {
>         lvalue = EvaluateExpTree(T->lchild);   // 递归计算左子树的值
>         rvalue = EvaluateExpTree(T->rchild);   // 递归计算右子树的值
>         return GetValue(T->data, lvalue, rvalue); // 按运算符类型运算
>     }
> }
> ```
> **【算法分析】** 求值是后序遍历过程，时间、空间复杂度均为 $O(n)$。
