## 1. 哈夫曼树的构造过程
![[Attachments/Pasted image 20260717185857.png]]
（1）给定 $n$ 个权值，构造 $n$ 棵只有根结点的二叉树，构成森林 $F$；
（2）在 $F$ 中选根权值最小的两棵树作左右子树，新树根权值为左右子树根权值之和；
（3）从 $F$ 删去这两棵树，加入新树；
（4）重复（2）（3）至 $F$ 只剩一棵树（即哈夫曼树）。

该过程总是先选权值小的，保证权值大的离根近，使 $WPL$ 最小，是一种典型**贪心法**（图 5.26）。

## 2. 哈夫曼算法的实现

哈夫曼树无度为 1 的结点，$n$ 个叶子的哈夫曼树共 $2n-1$ 个结点，可存于大小为 $2n-1$ 的一维数组；每结点含双亲与孩子信息（图 5.27）。

```cpp
//------ 哈夫曼树的存储表示 ------
typedef struct
{
    int weight;                 // 结点的权值
    int parent, lchild, rchild; // 结点的双亲、左孩子、右孩子的下标
} HTNode, *HuffmanTree;         // 动态分配数组存储哈夫曼树
```

数组 0 号单元不使用，从 1 号开始，大小为 $2n$；前 $n$ 个位置存叶子，后 $n-1$ 个位置存非叶子。

> [!example] 算法 5.10 构造哈夫曼树
> **【算法步骤】** ① 初始化：动态申请 $2n$ 单元，1~$2n-1$ 的 parent/lchild/rchild 全置 0，再输入前 $n$ 个叶子权值；② 创建树：循环 $n-1$ 次，每次选 parent 为 0 且权值最小的两棵树根 $s1,s2$，将其 parent 改为当前新结点下标，新结点左右孩子分别为 $s1,s2$，权值为二者之和。
> **【算法描述】**
> ```cpp
> void CreateHuffmanTree(HuffmanTree &HT, int n)
> { // 构造哈夫曼树 HT
>     if (n <= 1) return;
>     m = 2 * n - 1;
>     HT = new HTNode[m + 1];                     // 0 号单元未用，动态分配 m+1 个单元
>     for (i = 1; i <= m; ++i)                    // 双亲、左孩子、右孩子的下标初始化为 0
>     {
>         HT[i].parent = 0; HT[i].lchild = 0; HT[i].rchild = 0;
>     }
>     for (i = 1; i <= n; ++i)                    // 输入前 n 个叶子结点的权值
>         cin >> HT[i].weight;
>     /* ------ 初始化结束，下面创建哈夫曼树 ------ */
>     for (i = n + 1; i <= m; ++i)
>     { // 通过 n-1 次的选择、删除、合并来创建哈夫曼树
>         Select(HT, i - 1, s1, s2);              // 选 parent 为 0 且权值最小的两结点 s1,s2
>         HT[s1].parent = i; HT[s2].parent = i;   // 从森林删除 s1,s2，双亲改为 i
>         HT[i].lchild = s1; HT[i].rchild = s2;   // s1,s2 分别作为 i 的左右孩子
>         HT[i].weight = HT[s1].weight + HT[s2].weight; // i 的权值为左右孩子权值之和
>     }
> }
> ```
