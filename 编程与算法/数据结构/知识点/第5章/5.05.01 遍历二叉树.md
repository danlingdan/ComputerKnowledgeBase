## 1. 遍历二叉树算法描述

> [!definition] 遍历二叉树（traversing binary tree）
> 按某条搜索路径巡访树中每个结点，使每个结点均被访问一次且仅一次。遍历的实质是对二叉树进行**线性化**，将非线性结构的结点排成线性序列。

由二叉树的递归定义，它由根、左子树、右子树三个基本单元组成。用 L、D、R 分别表示遍历左子树、访问根、遍历右子树，限定先左后右则有三种：
- **先（根）序遍历（DLR）**：访问根 → 先序左子树 → 先序右子树；
- **中（根）序遍历（LDR）**：中序左子树 → 访问根 → 中序右子树；
- **后（根）序遍历（LRD）**：后序左子树 → 后序右子树 → 访问根。

对图 5.5 表示的表达式 $a + b * (c - d) - e / f$：
- 先序序列（前缀/波兰式）：$ - + a * b - c d / e f $（式 5-3）
- 中序序列（中缀）：$ a + b * c - d - e / f $（式 5-4）
- 后序序列（后缀/逆波兰式）：$ a b c d - * + e f / - $（式 5-5）

> [!example] 算法 5.1 中序遍历的递归算法
> **【算法描述】**
> ```cpp
> void InOrderTraverse(BiTree T)
> { // 中序遍历二叉树 T 的递归算法
>     if (T)                   // 若二叉树非空
>     {
>         InOrderTraverse(T->lchild);  // 中序遍历左子树
>         cout << T->data;             // 访问根结点
>         InOrderTraverse(T->rchild);  // 中序遍历右子树
>     }
> }
> ```
> 改变输出语句顺序即可得先序/后序递归算法。三种遍历的区别仅在于访问根与遍历左右子树的先后关系；从递归执行过程看，先序、中序、后序也完全相同（图 5.11 b 用虚线箭头表示递归调用/返回，三角形/圆形/方形分别表示先/中/后序访问时输出的信息）。

利用栈可将递归改写为非递归（类比 3.4.4 节）。
![[Attachments/Pasted image 20260717185558.png]]
> [!example] 算法 5.2 中序遍历的非递归算法
> **【算法步骤】**
> ① 初始化空栈 S，p 指向根；② 申请结点 q 存放栈顶弹出元素；③ 当 p 非空或栈 S 非空时循环：p 非空则 p 进栈、p 指向左孩子；p 为空则弹栈访问根、p 指向右孩子。
> **【算法描述】**
> ```cpp
> void InOrderTraverse(BiTree T)
> { // 中序遍历二叉树 T 的非递归算法
>     InitStack(S); p = T;
>     q = new BiTNode;
>     while (p || !StackEmpty(S))
>     {
>         if (p)                       // p 非空
>         {
>             Push(S, p);              // 根指针进栈
>             p = p->lchild;           // 遍历左子树
>         }
>         else                         // p 为空
>         {
>             Pop(S, q);               // 退栈
>             cout << q->data;         // 访问根结点
>             p = q->rchild;           // 遍历右子树
>         }
>     }                                // while
> }
> ```
> **【算法分析】** 含 $n$ 个结点的二叉树，不论哪种次序遍历，每个结点访问一次，时间复杂度均为 $O(n)$；辅助空间为栈的最大容量（树的深度），最坏 $O(n)$。

此外还有**层次遍历**（从上到下、从左到右，借助队列），图 5.11（a）的层次序列为 `-*cab`，此处不展开。

## 2. 根据遍历序列确定二叉树

> [!warning] 先序 + 后序不能唯一确定二叉树
> 由先序序列和中序序列，或后序序列和中序序列，均能唯一确定一棵二叉树；但**仅由先序序列和后序序列不能唯一确定**，因为无法区分左右子树（如先序 AB、后序 BA 可对应两棵不同二叉树，图 5.14）。

原理：先序序列第一个结点是根；中序序列中根将序列分为左子树中序与右子树中序，再在先序中找对应左右子序列，递归即可确定整棵树（同理后序最后一个结点是根）。

> [!example] 例 5.1
> 已知中序 BDCEAFHG、后序 DECBHGFA，求二叉树。
> （1）后序尾部为根 → A；（2）中序中 A 左侧 BDCE 为左子树、右侧 FHG 为右子树；（3）后序 DECB 确定 B 为 A 的左孩子，HGF 确定 F 为 A 的右孩子；递归得唯一二叉树（图 5.13）。

## 3. 遍历二叉树算法的应用

### （1）创建二叉树的存储结构——二叉链表

> [!example] 算法 5.3 按先序遍历的顺序建立二叉链表
> **【算法步骤】** 读取字符 ch；若 ch 为 “#” 则 T=NULL（空树），否则申请结点、ch 赋给 T->data、递归建左子树、递归建右子树。
> **【算法描述】**
> ```cpp
> void CreateBiTree(BiTree &T)
> { // 按先序次序输入结点值（单字符），创建二叉链表
>     cin >> ch;
>     if (ch == '#') T = NULL;             // 递归结束，建空树
>     else
>     {
>         T = new BiTNode;                 // 生成根结点
>         T->data = ch;                    // 根结点数据域置为 ch
>         CreateBiTree(T->lchild);         // 递归创建左子树
>         CreateBiTree(T->rchild);         // 递归创建右子树
>     }
> }
> ```
> 对图 5.10（b）的二叉树，读入 `ABC##DE#G##F###`（# 表示空树）可建立相应二叉链表。

### （2）复制二叉树

> [!example] 算法 5.4 复制二叉树
> **【算法步骤】** 空树则 NewT=NULL 返回；否则申请新结点复制根、递归复制左子树、递归复制右子树（与先序遍历结构一致）。
> **【算法描述】**
> ```cpp
> void Copy(BiTree T, BiTree &NewT)
> { // 复制一棵和 T 完全相同的二叉树
>     if (T == NULL)
>     {
>         NewT = NULL;
>         return;
>     }
>     else
>     {
>         NewT = new BiTNode;
>         NewT->data = T->data;           // 复制根结点
>         Copy(T->lchild, NewT->lchild);  // 递归复制左子树
>         Copy(T->rchild, NewT->rchild);  // 递归复制右子树
>     }
> }
> ```

### （3）计算二叉树的深度

> [!example] 算法 5.5 计算二叉树的深度
> **【算法步骤】** 空树深度 0；否则递归算左子树深度 $m$、右子树深度 $n$，返回 $\max(m,n)+1$。
> **【算法描述】**
> ```cpp
> int Depth(BiTree T)
> { // 计算二叉树 T 的深度
>     if (T == NULL) return 0;             // 空树深度为 0
>     else
>     {
>         m = Depth(T->lchild);            // 递归计算左子树的深度记为 m
>         n = Depth(T->rchild);            // 递归计算右子树的深度记为 n
>         if (m > n) return (m + 1);       // 深度为 m 与 n 的较大者加 1
>         else return (n + 1);
>     }
> }
> ```
> 计算深度是在**后序遍历**基础上进行的。

### （4）统计二叉树中结点的个数

> [!example] 算法 5.6 统计二叉树中结点的个数
> **【算法描述】**
> ```cpp
> int NodeCount(BiTree T)
> { // 统计二叉树 T 中结点的个数
>     if (T == NULL) return 0;                 // 空树结点个数为 0
>     else return NodeCount(T->lchild) + NodeCount(T->rchild) + 1;
>                                             // 左子树个数 + 右子树个数 + 1
> }
> ```
> 可类比写出统计叶子结点、度为 1、度为 2 结点个数的算法。
