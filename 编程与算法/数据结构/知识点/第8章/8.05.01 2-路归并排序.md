## 算法 8.10 相邻两个有序子序列的归并

> [!info] **算法步骤**
> 两有序表存于数组相邻位置 `R[low..mid]` 与 `R[mid+1..high]`，每次各取一记录比较，较小者放入 `T[low..high]`；重复至一表空，再将另一非空表余下部分复制到 T。

**【算法描述】**

```c
void Merge(RedType R[], RedType T[], int low, int mid, int high)
{ // 将有序表 R[low..mid] 和 R[mid+1..high] 归并为有序表 T[low..high]
    i=low; j=mid+1; k=low;
    while(i<=mid && j<=high)                // 将 R 中的记录由小到大并入 T 中
    {
        if(R[i].key<=R[j].key) T[k++]=R[i++];
        else T[k++]=R[j++];
    } // while
    while(i<=mid) T[k++]=R[i++];            // 将剩余的 R[i..mid] 复制到 T 中
    while(j<=high) T[k++]=R[j++];           // 将剩余的 R[j..high] 复制到 T 中
}
```

每子序列长 $h$，一趟需调用 $\lceil n/2h \rceil$ 次 `Merge`，整个排序需 $\lceil \log_2 n \rceil$ 趟。与快排类似，也可递归划分实现。

## 算法 8.11 归并排序

> [!info] **算法步骤**
> 将 `R[low..high]` 归并排序后放入 `T[low..high]`。长度等于 1 时递归结束，否则：① 一分为二，求 `mid = \lfloor (low+high)/2 \rfloor`；② 对 `R[low..mid]` 递归排序入 `S[low..mid]`；③ 对 `R[mid+1..high]` 递归排序入 `S[mid+1..high]`；④ 调 `Merge` 将两有序子序列归并入 `T[low..high]`。

**【算法描述】**

```c
void MSort(RedType R[], RedType T[], int low, int high)
{ // R[low..high] 归并排序后放入 T[low..high] 中
    if(low==high) T[low]=R[low];
    else
    {
        mid=(low+high)/2;                   // 将当前序列一分为二，求出分裂点 mid
        S=new RedType[MAXSIZE];
        MSort(R, S, low, mid);              // 对子序列 R[low..mid] 递归归并排序
        MSort(R, S, mid+1, high);           // 对子序列 R[mid+1..high] 递归归并排序
        Merge(S, T, low, mid, high);        // 将 S[low..mid] 和 S[mid+1..high] 归并到 T
    } // else
}

void MergeSort(SqList &L)
{ // 对顺序表 L 进行归并排序
    MSort(L.r, L.r, 1, L.length);
}
```

> [!abstract] **算法分析**
> **（1）时间复杂度**：$n$ 个记录需 $\lceil \log_2 n \rceil$ 趟，每趟比较不超过 $n$、移动均为 $n$，故 $O(n \log_2 n)$。
> **（2）空间复杂度**：顺序表实现需与记录数相等的辅助空间，为 $O(n)$。

> [!tip] **算法特点**
> （1）稳定排序。（2）可用于链式结构且不需附加存储空间（但递归仍需工作栈）。
