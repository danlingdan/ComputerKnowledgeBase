> [!definition] 冒泡排序（Bubble Sort）
> 最简单的交换排序。两两比较相邻记录关键字，若为逆序则交换，使关键字小的记录如气泡般上浮（左移），或大的记录如石块般下沉（右移）。

## 算法 8.4 冒泡排序

> [!info] **算法步骤**
> ① 比较 `r[1]` 与 `r[2]`，逆序则交换；再比 `r[2]` 与 `r[3]`……直至 `r[n-1]` 与 `r[n]`。第一趟使最大关键字落到最后位置。
> ② 第二趟对前 $n-1$ 个记录同样操作，使次大关键字落至第 $n-1$ 位。
> ③ 重复，第 $i$ 趟从 `r[1]` 到 `r[n-i+1]` 比较相邻记录并在逆序时交换；某趟未发生交换说明已有序，排序完成。

> [!example] **例 8.3** 序列 $\{49, 38, 65, 97, 76, 13, 27, 49\}$ 的冒泡排序过程。

**【算法描述】**

```c
void BubbleSort(SqList &L)
{ // 对顺序表 L 进行冒泡排序
    m=L.length-1; flag=1;                   // flag 用来标记某一趟排序是否发生交换
    while ((m>0) && (flag==1))
    {
        flag=0;                             // flag 置为 0，本趟无交换则不再执行下一趟
        for(j=1; j<=m; ++j)
            if(L.r[j].key > L.r[j+1].key)
            {
                flag=1;                     // flag 置为 1，表示本趟发生了交换
                t=L.r[j]; L.r[j]=L.r[j+1]; L.r[j+1]=t; // 交换前后两个记录
            }
        --m;
    } // while
} // BubbleSort
```

> [!abstract] **算法分析**
> **（1）时间复杂度**：最好情况（正序）仅一趟，$n-1$ 次比较、不移动；最坏情况（逆序）需 $n-1$ 趟，比较 $KCN$ 与移动 $RMN$（每次交换移动 3 次）分别为：
> $$ KCN = \sum_{i=1}^{n-1} (n-i) = \frac{n(n-1)}{2} \approx \frac{n^2}{2} $$
> $$ RMN = 3 \sum_{i=1}^{n-1} (n-i) = \frac{3n(n-1)}{2} \approx \frac{3n^2}{2} $$
> 平均比较约 $\frac{n^2}{4}$、移动约 $\frac{3n^2}{4}$，时间复杂度 $O(n^2)$。
> **（2）空间复杂度**：仅交换时需要一个辅助空间，为 $O(1)$。

> [!tip] **算法特点**
> （1）稳定排序。（2）可用于链式存储结构。（3）移动次数较多，平均时间性能比直接插入排序差；无序且 $n$ 大时不宜采用。
