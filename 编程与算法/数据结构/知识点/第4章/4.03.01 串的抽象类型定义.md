# 4.3.1 串的抽象类型定义

> [!topic] 所属主题
> [[MOC - 第4章 串、数组和广义表#4.3 串的类型定义、存储结构及其运算|4.3 串的类型定义、存储结构及其运算]]

串的逻辑结构和线性表极为相似，区别仅在于串的数据对象约束为字符集。然而，串的基本操作和线性表有很大差别：线性表多以“单个元素”为操作对象（查找/求取/插入/删除某元素），而串多以“串整体”为操作对象（查找子串、求取子串、插入/删除子串等）。

> [!info] 串的抽象数据类型定义
> ```text
> ADT String {
>     数据对象：D = { a_i | a_i ∈ CharacterSet, i = 1, 2, ..., n, n ≥ 0 }
>     数据关系：R1 = { <a_{i-1}, a_i> | a_{i-1}, a_i ∈ D, i = 2, ..., n }
>     基本操作：
>     StrAssign(&T, chars)     生成一个其值等于 chars 的串 T。
>     StrCopy(&T, S)           由串 S 复制得串 T。
>     StrEmpty(S)              若 S 为空串返回 true，否则 false。
>     StrCompare(S, T)         若 S>T 返回>0；S=T 返回 0；S<T 返回<0。
>     StrLength(S)             返回 S 的元素个数（串长）。
>     ClearString(&S)          将 S 清为空串。
>     Concat(&T, S1, S2)       用 T 返回由 S1 和 S2 连接而成的新串。
>     SubString(&Sub, S, pos, len)  返回 S 第 pos 个字符起长度为 len 的子串。
>     Index(S, T, pos)         返回 T 在 S 中第 pos 个字符之后首次出现的位置；否则 0。
>     Replace(&S, T, V)        用 V 替换 S 中所有与 T 相等的不重叠子串。
>     StrInsert(&S, pos, T)    在 S 第 pos 个字符之前插入串 T。
>     StrDelete(&S, pos, len)  从 S 中删除第 pos 个字符起长度为 len 的子串。
>     DestroyString(&S)        销毁串 S。
> } ADT String
> ```

对于串的基本操作集可以有不同的定义方法，读者在使用高级语言中的串类型时，应以相应语言的参考手册为准。
