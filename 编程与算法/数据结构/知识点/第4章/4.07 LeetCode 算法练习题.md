## 【算法练习题 4.1】LeetCode 125 验证回文串 ★

【问题描述】
给定字符串 `s`，判断是否为回文串：将所有大写字母转小写、移除所有非字母数字字符后，正读反读相同的字符串。字母和数字均属字母数字字符。

【输入输出示例】
输入：`s = "A man, a plan, a canal: Panama"`　输出：`true`（`"amanplanacanalpanama"` 是回文串）。

【问题分析】
在原字符串 `s` 上用 `left`、`right` 双指针：初始分别指向首尾；若所指为非字母数字字符则向内移动，直至指向字母/数字；比较转为小写后是否相同，相同则继续，直至 `left >= right` 返回 `true`，否则返回 `false`。步骤如图 4.23。

![[Attachments/Pasted image 20260717172009.png]]
（图4.23 验证回文串双指针实现步骤）

【算法步骤】
① `left` 指向首、`right` 指向尾。
② 当 `left < right`：若 `left` 所指非字母数字则右移；若 `right` 所指非字母数字则左移；若转小写后不同则返回 `false`；相同则 `left++`、`right--`。
③ `left >= right` 返回 `true`。

【算法描述】

```c
bool isPalindrome(char * s)
{ // 验证回文串
    int n = strlen(s);
    int left = 0, right = n - 1;
    while (left < right)                 // 遍历字符串 s
    {
        while (left < right && !isalnum(s[left]))   // left 所指字符为非字母数字字符
            ++left;                                 // 右移一步
        while (left < right && !isalnum(s[right]))  // right 所指字符为非字母数字字符
            --right;                                // 左移一步
        if (left < right)
        {
            if (tolower(s[left]) != tolower(s[right]))  // 所指字符不同，返回 false
                return false;
            ++left;                                     // 所指字符相同，移动 left 和 right
            --right;
        }
    }
    return true;                                        // left >= right，返回 true
}
```

> [!example] 练习题 4.1 验证回文串
> 时间复杂度 $O(n)$（最坏遍历每个字符），空间复杂度 $O(1)$（两指针）。

## 【算法练习题 4.2】LeetCode 566 重塑矩阵 ★

【问题描述】
MATLAB 的 `reshape` 可将 $m \times n$ 矩阵重塑为 $r \times c$ 新矩阵并保留数据。给定 `mat`（$m \times n$）、正整数 $r,c$，按相同行遍历顺序填充；若可行合理则输出重塑矩阵，否则输出原矩阵。

【输入输出示例】
输入：`mat = [[1,2],[3,4]]`，`r = 1, c = 4`　输出：`[[1,2,3,4]]`（图 4.24）。

![[Attachments/Pasted image 20260717172018.png]]
（图4.24 矩阵重塑示意）

【问题分析】
先判断 $m \times n$ 与 $r \times c$ 是否相等，不等直接返回原矩阵；相等则可映射：一维数组中第 $x \in [0, m \times n)$ 个元素，在 `mat` 中下标 $(x / n, x \% n)$，在重塑矩阵 `ans` 中下标 $(x / c, x \% c)$，直接赋值即可。

【算法步骤】
① 初始化，得原矩阵行数、列数。
② 若 $m \times n \neq r \times c$，返回原矩阵。
③ 若相等，初始化重塑矩阵，对 $x \in [0, m \times n)$ 循环：`ans[x/c][x%c] = mat[x/n][x%n]`。
④ 返回 `ans`。

【算法描述】

```c
int** matrixReshape(int** mat, int matSize, int* matColSize, int r, int c,
                    int* returnSize, int** returnColumnSizes)
{ // 重塑矩阵
    int m = matSize;
    int n = matColSize[0];
    if (m * n != r * c)                  // 给定参数的 reshape 操作不合理，返回原矩阵
    {
        *returnSize = matSize;
        *returnColumnSizes = matColSize;
        return mat;
    }
    *returnSize = r;
    *returnColumnSizes = (int*)malloc(r * sizeof(int));
    int** ans = (int**)malloc(r * sizeof(int));
    for (int i = 0; i < r; i++)          // 对重塑的新矩阵进行初始化
    {
        (*returnColumnSizes)[i] = c;
        ans[i] = (int*)malloc(c * sizeof(int));
    }
    for (int x = 0; x < m * n; ++x)      // 矩阵重塑
        ans[x / c][x % c] = mat[x / n][x % n];
    return ans;
}
```

> [!example] 练习题 4.2 重塑矩阵
> 重塑成功时时间复杂度 $O(rc)$，否则直接返回 $O(1)$；空间复杂度 $O(1)$（不额外申请与结果无关的空间）。

## 【算法练习题 4.3】LeetCode 3 无重复字符的最长子串 ★★

【问题描述】
给定字符串 `s`，找出不含有重复字符的最长子串的长度。

【输入输出示例】
输入：`s = "abcabcbb"`　输出：`3`（无重复最长子串 `"abc"`）。

【问题分析】
用滑动窗口思想，`left`、`right` 表示窗口左右端下标：检测 `left` 到 `right` 子串是否含重复；无重复时 `right` 右移并记录最长长度；出现重复时 `left` 移至左侧重复字符下一位置；返回 `max`。步骤如图 4.25。

![[Attachments/Pasted image 20260717172027.png]]
（图4.25 滑动窗口求最长无重复子串步骤）

【算法步骤】
① `left`、`right`、`max` 初值 0。
② 遍历 `s`：当 `left <= right`，在子串中查找重复字符，若重复则 `left` 移至下一位置；否则更新最长长度，`right` 右移。
③ 返回 `max`。

【算法描述】

```c
int lengthOfLongestSubstring(char * s)
{ // 返回无重复字符的最长子串的长度
    int left = 0, right = 0, max = 0;
    int len = strlen(s);
    for (int i = 0; i < len; i++ )
    {
        if (left <= right)
        {
            for (int j = left; j < right ; j++)   // 在 left 和 right 之间查找重复字符
            {
                if (s[j] == s[right])             // 检测是否出现重复
                {
                    left = j + 1;
                    break;
                }
            }
        }
        max = max < (right - left + 1) ? (right - left + 1) : max; // 更新最长子串长度
        right++;
    }
    return max;
}
```

> [!example] 练习题 4.3 无重复字符的最长子串
> 最坏情况外层遍历全长、内层遍历 `[left, right]`（最长 $n$），时间复杂度 $O(n^2)$；空间复杂度 $O(1)$。

## 【算法练习题 4.4】LeetCode 6 Z 字形变换 ★★

【问题描述】
将字符串 `s` 按给定行数 `numRows`，从上向下、从左到右进行 Z 字形排列，再从左往右逐行读取产生新字符串输出。如 `PAYPALISHIRING`、行数 3 排列后输出 `"PAHNAPLSIIGYIR"`。

【输入输出示例】
输入：`s = "PAYPALISHIRING", numRows = 4`　输出：`"PINALSIGYAHRPI"`。

【问题分析】
若行数 $r = 1$ 或 $r \geqslant n$，新串与原串相同直接返回；否则创建二维矩阵按 Z 字形填 `s`，最后逐行读非空字符。变换周期 $t = 2r - 2$。可用压缩矩阵 `mat`（每行一空列表）存储：计算下一元素所在行，若 $i \% t < r - 1$ 则下一行为 `x` 的下一行，否则为上一行。输入输出示例对应压缩矩阵如图 4.26。

![[Attachments/Pasted image 20260717172037.png]]
（图4.26 Z字形变换的压缩矩阵存储示意）

【算法步骤】
① 若 $r = 1$ 或 $r \geqslant n$，返回 `s`。
② 初始化压缩矩阵 `mat`。
③ 遍历 `s`：将 `s[i]` 追加到 `mat` 第 `x` 行列表尾；计算下一行（若 $i \% t < r - 1$ 则 `x` 下一行，否则上一行）。
④ 遍历 `mat`，将元素赋值给 `s`。
⑤ 返回新字符串 `s`。

【算法描述】

```c
char* convert(char* s, int numRows) {
    int n = strlen(s), r = numRows;
    if (r == 1 || r >= n)
        return s;
    char** mat = (char**)malloc(r * sizeof(char*));
    int* colSize = (int*)malloc(numRows * sizeof(int));
    memset(colSize, 0, sizeof(int) * numRows);
    for (int i = 0; i < r; i++)              // 矩阵初始化
    {
        mat[i] = (char*)malloc(n * sizeof(char));
        memset(mat[i], 0, sizeof(char) * (n));
    }
    for (int i = 0, x = 0, t = r * 2 - 2; i < n; ++i)  // 填充压缩矩阵
    {
        mat[x][colSize[x]++] = s[i];         // 将 s[i] 放入矩阵当前位置
        i % t < r - 1 ? ++x : --x;           // 计算下一个元素所在行
    }
    int pos = 0;
    for (int i = 0; i < r; i++)              // 逐行遍历压缩矩阵
    {
        for (int j = 0; j < colSize[i]; j++)
            s[pos++] = mat[i][j];            // 从矩阵读取字符并写回原字符串
        free(mat[i]);
    }
    free(colSize);
    free(mat);
    return s;
}
```

> [!example] 练习题 4.4 Z 字形变换
> 初始化矩阵 $O(r \times n)$，填充遍历 $O(n)$，逐行读取 $O(n)$，一般 $r \ll n$，整体时间复杂度 $O(n)$；空间复杂度 $O(n)$（压缩矩阵）。

## 【算法练习题 4.5】LeetCode 443 压缩字符串 ★★

【问题描述】
给定字符数组 `chars`，原地压缩：对每组连续重复字符，长度 1 则追加该字符，否则追加字符后跟组长度；组长度 ≥10 时拆为多个字符。压缩后转存回 `chars`，返回新长度，要求常量额外空间。

【输入输出示例】
输入：`chars = ["a","b","b",...,"b"]`（13 个 b）　输出：`4`（压缩为 `["a","b","1","2"]`）。

【问题分析】
用 `write`、`read` 标识读写位置。对每组连续相同字符，`read` 移到最右侧，在 `chars[write]` 写 `chars[read]` 并记录长度 `num = read - left + 1`（`left` 记录子串最左）。`num = 1` 则 `left` 右移；`num > 1` 用短除法将 `num` 倒序写入 `chars` 再反转。返回 `write`。步骤如图 4.27。

![[Attachments/Pasted image 20260717172044.png]]
（图4.27 原地压缩字符串的步骤示意）

【算法步骤】
① `write`、`left` 初值 0。
② 遍历 `chars`：当 `read` 在末或所指字符不同于下一字符时，写字符到 `chars[write++]`，记录 `num`；若 `num > 1` 用短除法倒序写入并反转；`left` 移到下一组最左。
③ 返回 `write`。

【算法描述】

```c
void swap(char *a, char *b)
{ // 交换数值
    char t = *a;
    *a = *b, *b = t;
}
void reverse(char *a, char *b)
{ // 反转字符数组
    while (a < b)
        swap(a++, --b);
}
int compress(char *chars, int charsSize)
{ // 返回压缩字符串长度
    int write = 0, left = 0;
    for (int read = 0; read < charsSize; read++)  // 遍历字符数组 chars
    {
        if (read == charsSize - 1 || chars[read] != chars[read + 1])
        { // read 位于字符串末尾或所指字符不同于下一个字符
            chars[write++] = chars[read];
            int num = read - left + 1;            // 记录子串长度
            if (num > 1)                          // 子串长度大于 1
            {
                int anchor = write;
                while (num > 0)                   // 数字转化为字符串倒序写入原字符串
                {
                    chars[write++] = num % 10 + '0';
                    num /= 10;
                }
                reverse(&chars[anchor], &chars[write]); // 反转
            }
            left = read + 1;                      // left 移到下一组字符的起始位置
        }
    }
    return write;
}
```

> [!example] 练习题 4.5 压缩字符串
> 只需遍历一次输入，时间复杂度 $O(n)$；仅用常数个变量，空间复杂度 $O(1)$。
