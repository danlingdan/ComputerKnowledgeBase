## 算法练习题 7.1 LeetCode 235 二叉搜索树的最近公共祖先 ★★

> [!example] LeetCode 235
> 【问题描述】给定一棵二叉排序树（二叉搜索树），求两个指定结点 $p$、$q$ 的**最近公共祖先**（最深、且 $p$、$q$ 之一也可为自己的祖先）。
> 输入：`root = [6, 2, 8, 0, 4, 7, 9, NULL, NULL, 3, 5]`，`p=2`，`q=8`；输出：`6`。
> ![[Attachments/Pasted image 20260718194433.png]]
> ![[Attachments/Pasted image 20260718194440.png]]
> ![[Attachments/Pasted image 20260718194447.png]]
> **图 7.57 二叉排序树**（树根 6，左孩子 2，右孩子 8）　**图 7.58 LeetCode 235 实现步骤**（a 初始位置；b 从根遍历；c 值大于 p,q 遍历左；d 值小于 p,q 遍历右；e 找到最近公共祖先）
>
> 【算法步骤】① `ancestor` 初始指向根；② 若 `ancestor` 值同时大于 $p,q$ 则向左，同时小于则向右，否则跳出；③ 返回 `ancestor`。
> 【算法描述】
> ```c
> typedef struct TreeNode
> { // 二叉树的二叉链表存储结构
>     int val;                        // 结点值
>     struct TreeNode *left, *right;  // 左右孩子指针
> }TreeNode, * BiTree;
>
> struct TreeNode* lowestCommonAncestor(struct TreeNode* root, struct TreeNode* p, struct TreeNode* q)
> { // 查找二叉排序树的最近公共祖先
>     struct TreeNode* ancestor = root;       // 从根结点开始遍历
>     while (true)
>     {
>         if (ancestor->val > p->val && ancestor->val > q->val)
>             ancestor = ancestor->left;      // 大于p和q，遍历左子树
>         else if (ancestor->val < p->val && ancestor->val < q->val)
>             ancestor = ancestor->right;     // 小于p和q，遍历右子树
>         else
>             break;
>     }
>     return ancestor;                        // 返回最近公共祖先
> }
> ```
> 【算法分析】时间与 $p,q$ 深度线性相关，最坏 $O(n)$（链状树），空间 $O(1)$。

## 算法练习题 7.2 LeetCode 34 在排序数组中查找元素的第一个和最后一个位置 ★★

> [!example] LeetCode 34
> 【问题描述】给定非递减整数数组 `nums` 与目标值 `target`，找出 `target` 的开始和结束位置；不存在则返回 `[-1, -1]`，时间复杂度 $O(\log_2 n)$。
> 输入：`nums = [5, 7, 7, 8, 8, 10]`，`target = 8`；输出：`[3, 4]`。
> ![[Attachments/Pasted image 20260718194457.png]]
> ![[Attachments/Pasted image 20260718194506.png]]
> **图 7.59 LeetCode 34 实现步骤**（a 初始；b mid；c nums[mid]<target 令 left=mid+1；… e 得 leftIdx；h~n 第二次查找得 rightIdx 并返回）
>
> 【算法步骤】① 调用 `binarySearch(nums, target, lower)` 确定左右边界 `leftIdx`、`rightIdx`；② 定义返回数组 `ret`；③ 若 `leftIdx<=rightIdx && rightIdx<长度 && 两端均等于 target`，返回边界，否则返回 `[-1,-1]`。
> 【算法描述】
> ```c
> int binarySearch(int* nums, int numsSize, int target, bool lower)
> { // 二分查找，返回nums数组中二分查找target的位置
>     int left=0, right=numsSize-1, ans=numsSize;
>     while (left<=right)
>     {
>         int mid=(left+right)/2;
>         if (nums[mid]>target || (lower && nums[mid]>=target))
>         // 返回nums数组中第一个等于target的位置或第一个大于target的位置
>             right=mid-1;
>             ans=mid;
>         }
>         else
>             left=mid+1;
>     }
>     return ans;
> }
>
> int* searchRange(int* nums, int numsSize, int target, int* returnSize)
> { // 查找给定目标值在数组中的开始位置和结束位置
>     int leftIdx=binarySearch(nums, numsSize, target, true);
>     int rightIdx=binarySearch(nums, numsSize, target, false)-1;
>     int* ret=new int[sizeof(int)*2];
>     *returnSize=2;                      // 返回数组ret的元素个数为2
>     if (leftIdx<=rightIdx && rightIdx<numsSize && nums[leftIdx]==target && nums[rightIdx]==target)
>     { // 存在目标值且对应元素均为target
>         ret[0]=leftIdx, ret[1]=rightIdx;
>         return ret;
>     }
>     ret[0]=-1, ret[1]=-1;               // 不存在目标值，返回[-1,-1]
>     return ret;
> }
> ```
> 【算法分析】二分查找 $O(\log_2 n)$，执行两次，总 $O(\log_2 n)$；空间 $O(1)$。

## 算法练习题 7.3 LeetCode 153 寻找旋转排序数组中的最小值 ★★

> [!example] LeetCode 153
> 【问题描述】升序数组经 1~n 次旋转后得到输入数组（元素互不相同），写 $O(\log_2 n)$ 算法返回最小元素。
> 输入：`nums = [3, 4, 5, 1, 2]`；输出：`1`（原数组 `[1,2,3,4,5]` 旋转 3 次）。
> ![[Attachments/Pasted image 20260718194517.png]]
> **图 7.60 经过旋转后的数组折线图**（标注最小值位置）
>
> 【问题分析】升序数组旋转后，最小值右侧元素均严格小于末元素 $x$，左侧元素均严格大于 $x$。用 `low`、`high` 表示区间，`mid` 为中点：若 `nums[mid] < nums[high]`，最小值在左半；否则在右半。
> 【算法步骤】① `low=0, high=n-1`；② 循环：`mid=(low+high)/2`；若 `nums[mid]<nums[high]` 则 `high=mid`，否则 `low=mid+1`；③ 返回 `nums[low]`。
> 【算法描述】
> ```c
> int findMin(int* nums, int numsSize)
> { // 查找经过旋转后的数组中的最小元素
>     int low = 0;
>     int high = numsSize - 1;
>     while (low < high) {
>         int mid = (low + high) / 2;
>         if (nums[mid] < nums[high])
>             high = mid;
>         else
>             low = mid + 1;
>     }
>     return nums[low];
> }
> ```
> 【算法分析】每步忽略一半区间，时间 $O(\log_2 n)$；空间 $O(1)$。
