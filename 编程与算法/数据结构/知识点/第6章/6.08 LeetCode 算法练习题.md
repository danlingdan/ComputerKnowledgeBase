> [!example] 算法练习题 6.1 LeetCode 743 网络延迟时间 ★★
> **【问题描述】** 含 $n$ 个结点的网络，信号经有向边传递时间用 `times` 表示（`times[i]=(u_i,v_i,w_i)`）。从结点 $k$ 发信号，多久使所有结点收到？不能则返 $-1$。
> **【输入输出示例】** 输入 `times=[[2,1,1],[2,3,1],[3,4,1]]`, $n=4$, $k=2$ → 输出 2（图 6.30）。
> **【问题分析】** 单源最短路径（Dijkstra）。从 $k$ 到其余各点最短路径长度的最大值即答案；存在不可达点则返 $-1$。
> **【算法描述】**
> ```c
> int networkDelayTime(int** times, int timesSize, int* timesColSize, int n, int k)
> {
>     const int inf = 0x3f3f3f3f;
>     int** g = (int**)malloc(n * sizeof(int*));
>     for (int i = 0; i < n; i++) {
>         g[i] = (int*)malloc(n * sizeof(int));
>         memset(g[i], inf, sizeof(int) * n);
>     }
>     for (int i = 0; i < timesSize; i++) {
>         int v1 = times[i][0], v2 = times[i][1];
>         g[v1 - 1][v2 - 1] = times[i][2];
>     }
>     int* dist = (int*)malloc(n * sizeof(int));
>     memset(dist, inf, sizeof(int) * n);
>     dist[k - 1] = 0;
>     int* used = (int*)malloc(n * sizeof(int));
>     memset(used, 0, sizeof(int) * n);
>     int ans = 0;
>     for (int i = 0; i < n; i++) {
>         int x = -1;
>         for (int y = 0; y < n; ++y)
>             if (!used[y] && (x == -1 || dist[y] < dist[x]))
>                 x = y;
>         used[x] = true;
>         ans = fmax(ans, dist[x]);
>         for (int y = 0; y < n; ++y)
>             dist[y] = fmin(dist[y], dist[x] + g[x][y]);
>     }
>     return ans == inf ? -1 : ans;
> }
> ```
> **【算法分析】** 时间复杂度 $O(n^2+m)$，空间复杂度 $O(n^2)$（邻接矩阵）。

> [!example] 算法练习题 6.2 LeetCode 547 省份数量 ★★
> **【问题描述】** $n$ 个城市，矩阵 `isConnected[i][j]=1` 表示 $i,j$ 直接相连。返回省份（连通分量）数目。
> **【输入输出示例】** 输入 `[[1,1,0],[1,1,0],[0,0,1]]` → 输出 2（图 6.31）。
> ![[Attachments/Pasted image 20260717192842.png]]
> **【问题分析】** `isConnected` 即邻接矩阵；省份即连通分量，可用 DFS 统计。
> **【算法描述】**
> ```c
> void DFS(int** isConnected, int* visited, int cities, int i)
> {
>     for (int j = 0; j < cities; j++)
>         if (isConnected[i][j] == 1 && !visited[j])
>         {
>             visited[j] = 1;
>             DFS(isConnected, visited, cities, j);
>         }
> }
> int findCircleNum(int** isConnected, int isConnectedSize, int* isConnectedColSize)
> {
>     int cities = isConnectedSize;
>     int* visited = (int*)malloc(cities * sizeof(int));
>     memset(visited, 0, sizeof(int) * cities);
>     int provinces = 0;
>     for (int i = 0; i < cities; i++)
>         if (!visited[i]) { DFS(isConnected, visited, cities, i); provinces++; }
>     return provinces;
> }
> ```
> **【算法分析】** 时间复杂度 $O(n^2)$，空间复杂度 $O(n)$。

> [!example] 算法练习题 6.3 LeetCode 841 钥匙和房间 ★★
> **【问题描述】** $n$ 个房间，0 号初始可进。进入房间 $i$ 可获得其中钥匙集合 `rooms[i]`，能打开对应房间。给定 `rooms`，能否进入所有房间？
> **【输入输出示例】** 输入 `rooms=[[1],[2],[3],[]]` → 输出 `true`。
> **【问题分析】** 把房间看作有向图顶点，$x$ 中有 $y$ 的钥匙即边 $\langle x,y\rangle$；问题转化为从顶点 0 出发能否到达所有顶点（BFS）。
> **【算法描述】**
> ```c
> bool canVisitAllRooms(int** rooms, int roomsSize, int* roomsColSize)
> {
>     int* visited = (int*)malloc(roomsSize * sizeof(int));
>     memset(visited, 0, sizeof(int) * roomsSize);
>     int* que = (int*)malloc(roomsSize * sizeof(int));
>     memset(que, 0, sizeof(int) * roomsSize);
>     int left = 0, right = 1, num = 0;
>     visited[0] = true; que[0] = 0;
>     while (left < right) {
>         int x = que[left++];
>         num++;
>         for (int i = 0; i < roomsColSize[x]; i++) {
>             if (!visited[rooms[x][i]]) {
>                 visited[rooms[x][i]] = true;
>                 que[right++] = rooms[x][i];
>             }
>         }
>     }
>     if (num == roomsSize) return true;
>     else return false;
> }
> ```
> **【算法分析】** 时间复杂度 $O(n+m)$，空间复杂度 $O(n)$（$n$ 房间数，$m$ 钥匙总数）。
