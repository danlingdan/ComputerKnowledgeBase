> [!definition] 最小生成树（MST）
> 连通网中各生成树里**各边代价之和最小**的那棵，称为最小代价生成树（Minimum Cost Spanning Tree），简称最小生成树。
> **MST 性质**：设 $N=(V,E)$ 为连通网，$U\subseteq V$ 非空；若 $(u,v)$ 是所有 $u\in U, v\in V-U$ 的边中权值最小的，则必存在一棵包含 $(u,v)$ 的最小生成树（可用反证法证明）。

Prim 与 Kruskal 均利用 MST 性质构造最小生成树。

## 1. 普里姆算法

![[Attachments/Pasted image 20260717192534.png]]

> [!example] 算法 6.8 普里姆算法（Prim，加点法）
> **【算法步骤】**
> ① 将初始顶点 $u$ 加入 $U$，对其余顶点 $v_j$ 初始化 `closedge[j]`。
> ② 循环 $n-1$ 次：从 `closedge` 选最小边 `closedge[k]`，输出此边，$k$ 并入 $U$，更新剩余各组的 `closedge[j]`。
> **【辅助数组】** `closedge[i-1]` 含两域：`lowcost`（到 $U$ 的最小边权值）、`adjvex`（该边在 $U$ 中的顶点）。
> ```c
> struct
> {
>     VerTexType adjvex;
>     ArcType lowcost;
> }closedge[MVNum];
> ```
> **【算法描述】**
> ```c
> void MiniSpanTree_Prim(AMGraph G, VerTexType u)
> { // 无向网 G 以邻接矩阵形式存储，从顶点 u 出发构造 G 的最小生成树，输出树的各条边
>     k=LocateVex(G, u);
>     for(j=0; j<G.vexnum; ++j)
>         if(j!=k) closedge[j]={u, G.arcs[k][j]};
>     closedge[k].lowcost=0;               // 初始，U={u}
>     for(i=1; i<G.vexnum; ++i)
>     {
>         k=Min(closedge);
>         u0=closedge[k].adjvex; v0=G.vexs[k];
>         cout<<u0<<v0;
>         closedge[k].lowcost=0;
>         for(j=0; j<G.vexnum; ++j)
>             if(G.arcs[k][j] < closedge[j].lowcost)
>                 closedge[j]={G.vexs[k], G.arcs[k][j]};
>     }//for
> }
> ```
> **【算法分析】** 时间复杂度 $O(n^2)$，与边数无关，适用于**稠密网**。

## 2. 克鲁斯卡尔算法

![[Attachments/Pasted image 20260717192545.png]]

> [!example] 算法 6.9 克鲁斯卡尔算法（Kruskal，加边法）
> **【算法步骤】**
> ① 将边按权值从小到大排序。
> ② 依次选最小边，若其两顶点落在不同连通分量（不形成回路）则加入 $T$，否则舍去；重复至所有顶点同属一个分量。
> **【辅助结构】** `Edge[]`（边信息：`Head`、`Tail`、`lowcost`）；`Vexset[i]`（顶点所属连通分量，初值 $i$）。
> ```c
> struct { VerTexType Head; VerTexType Tail; ArcType lowcost; }Edge[arcnum];
> int Vexset[MVNum];
> ```
> **【算法描述】**
> ```c
> void MiniSpanTree_Kruskal(AMGraph G)
> {
>     Sort(Edge);
>     for(i=0; i<G.vexnum; ++i) Vexset[i]=i;
>     for(i=0; i<G.arcnum; ++i)
>     {
>         v1=LocateVex(G, Edge[i].Head); v2=LocateVex(G, Edge[i].Tail);
>         vs1=Vexset[v1]; vs2=Vexset[v2];
>         if(vs1!=vs2)
>         {
>             cout<< Edge[i].Head << Edge[i].Tail;
>             for(j=0; j<G.vexnum; ++j)
>                 if(Vexset[j]==vs2) Vexset[j]=vs1;
>         }//if
>     }//for
> }
> ```
> **【算法分析】** 若用堆排序，时间复杂度 $O(e\log_2 e)$，与边数有关，适用于**稀疏网**。
