# 6.3 图的类型定义

> [!nav] 导航
> 上一知识点：[[6.02 案例引入]] · [[MOC - 第6章 图|本章目录]] · [[MOC - 数据结构|课程总览]] · 下一知识点：[[6.04.01 邻接矩阵]]

图加一组基本操作构成抽象数据类型。

> [!info] ADT Graph（图）
> ```c
> ADT Graph{
>     数据对象：V 是具有相同特性的数据元素的集合，称为顶点集。
>     数据关系：
>         R = {VR}
>         VR = {<v, w> | v, w ∈ V 且 P(v, w) <v, w> 表示从 v 到 w 的弧，
>         谓词 P(v, w) 定义了弧 <v, w> 的意义或信息}
>     基本操作：
>         CreateGraph(&G, V, VR)    // 按 V 和 VR 构造图 G
>         DestroyGraph(&G)          // 销毁图 G
>         LocateVex(G, u)           // 返回顶点 u 在图中的位置
>         GetVex(G, v)              // 返回 v 的值
>         PutVex(&G, v, value)      // 对 v 赋值 value
>         FirstAdjVex(G, v)         // 返回 v 的第一个邻接顶点
>         NextAdjVex(G, v, w)       // 返回 v 相对于 w 的下一个邻接顶点
>         InsertVex(&G, v)          // 增添新顶点 v
>         DeleteVex(&G, v)          // 删除顶点 v 及其相关弧
>         InsertArc(&G, v, w)       // 增添弧 <v, w>（无向图同时增添对称弧）
>         DeleteArc(&G, v, w)       // 删除弧 <v, w>（无向图同时删除对称弧）
>         DFSTraverse(G)            // 深度优先遍历
>         BFSTraverse(G)            // 广度优先遍历
> }ADT Graph
> ```
