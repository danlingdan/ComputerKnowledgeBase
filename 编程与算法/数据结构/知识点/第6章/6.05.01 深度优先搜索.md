## 2. 深度优先搜索遍历的算法实现

> [!example] 算法 6.3 深度优先搜索遍历连通图
> **【算法描述】**
> ```c
> bool visited[MVNum];                     // 访问标志数组，其初值为 "false"
> void DFS(Graph G, int v)
> { // 从第 v 个顶点出发递归地深度优先遍历图 G
>     cout<<v; visited[v]=true;            // 访问第 v 个顶点，并置访问标志数组相应分量值为 true
>     for (w=FirstAdjVex(G, v); w>=0; w=NextAdjVex(G, v, w))
>         if (!visited[w]) DFS(G, w);      // 对 v 的尚未访问的邻接顶点 w 递归调用 DFS()
> }
> ```

非连通图需循环调用（算法 6.4）：

> [!example] 算法 6.4 深度优先搜索遍历非连通图
> **【算法描述】**
> ```c
> void DFSTraverse(Graph G)
> { // 对非连通图 G 进行深度优先遍历
>     for(v=0; v<G.vexnum; ++v) visited[v]=false;    // 访问标志数组初始化
>     for(v=0; v<G.vexnum; ++v)                      // 循环调用算法 6.3
>         if (!visited[v]) DFS(G, v);                // 对尚未访问的顶点调用 DFS
> }
> ```
> 每调用一次算法 6.3 遍历一个连通分量，调用次数即连通分量数。

> [!example] 算法 6.5 采用邻接矩阵表示图的深度优先搜索遍历
> **【算法描述】**
> ```c
> void DFS_AM(AMGraph G, int v)
> {
>     cout<<v; visited[v]=true;
>     for(w=0; w<G.vexnum; w++)
>         if((G.arcs[v][w]!=0)&&(!visited[w])) DFS_AM(G, w);
> }
> ```

> [!example] 算法 6.6 采用邻接表表示图的深度优先搜索遍历
> **【算法描述】**
> ```c
> void DFS_AL(ALGraph G, int v)
> {
>     cout<<v; visited[v]=true;
>     p=G.vertices[v].firstarc;
>     while(p!=NULL)
>     {
>         w=p->adjvex;
>         if(!visited[w]) DFS_AL(G, w);
>         p=p->nextarc;
>     }//while
> }
> ```

> [!note] 深度优先搜索遍历的算法分析
> 每个顶点至多调用一次 `DFS()`，遍历实质是对每个顶点查找邻接点。邻接矩阵 $O(n^2)$；邻接表 $O(n+e)$。
