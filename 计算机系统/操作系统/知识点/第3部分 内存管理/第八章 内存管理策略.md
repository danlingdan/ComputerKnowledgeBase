# 第八章 内存管理策略

> [!abstract] 本章解决什么问题？
> 程序必须加载到内存才能执行，但内存是有限的稀缺资源。本章系统介绍内存管理的核心技术：从基本的地址绑定、逻辑/物理地址分离，到动态加载/链接、交换机制，再到连续内存分配（固定/可变分区）、分段和分页，最后深入页表结构（多级页表、哈希页表、倒置页表）及 Intel/ARM 架构实现。

> [!abstract] 章节定位
> 本章已按稳定知识对象拆分为 8 篇主题笔记。先沿下表建立机制主线，再进入具体主题查看定义、状态、算法、实现边界与例子。

## 学习目标

- [ ] 能区分逻辑地址与物理地址，解释 MMU 的作用。
- [ ] 能描述地址绑定的三种时机及其特点。
- [ ] 能解释动态加载与动态链接的区别与联系。
- [ ] 能描述连续内存分配的分区策略和分配算法。
- [ ] 能区分内部碎片与外部碎片及解决方案。
- [ ] 能解释分段与分页的区别及各自优缺点。
- [ ] 能描述 TLB 的工作原理和有效访问时间计算。
- [ ] 能解释多级页表、哈希页表和倒置页表的设计目的。
- [ ] 能描述 x86-64 四级分页机制。
