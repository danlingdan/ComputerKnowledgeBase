# 2.5 系统程序

本节聚焦于**系统程序**，是[[第二章 操作系统结构]]中的独立知识节点。

> [!summary] 一句话定义
> 系统程序位于**操作系统之上、应用程序之下**，为上层应用程序和系统管理员提供方便开发与执行的实用环境。

## 七大类核心功能

| 类别 | 具体功能 |
| --- | --- |
| 文件管理 | 文件与目录的增删改查、复制、打印等操作。 |
| 状态信息 | 获取日期时间、内存/磁盘使用量、用户登录数、系统性能、调试信息，支持注册表配置。 |
| 文件修改 | 文本编辑器、文件内容查找、格式转换工具。 |
| 程序语言支持 | 各种编程语言的编译、汇编、调试、解释器工具链。 |
| 程序加载与执行 | 各类加载器（绝对/重定位）、链接编辑器。 |
| 通信 | 进程间、用户间、机器间的虚拟连接（消息发送、网页浏览、邮件传输、远程登录、文件传送）。 |
| 后台服务 | 引导时启动的**守护进程（Daemon）/服务（Service）**，如网络监听、进程调度、监控、打印服务等。 |
