# 第十五章 系统安全

系统安全（system security）研究攻击者可能如何突破系统边界，以及操作系统如何通过认证、密码学、审计、纵深防御与持续修补降低风险。它与[[第十四章 系统保护|系统保护]]共同回答“谁可以做什么”，但更强调恶意行为、外部威胁和信任建立。

> [!warning] 版本边界
> 本章保留教材的经典威胁模型和密码学框架；具体算法、协议与平台支持会快速变化，部署时必须依据当前标准和厂商文档。

## 学习目标

- [ ] 理解安全违规的五个主要类型（机密、诚信、可用、盗窃服务、拒绝服务）
- [ ] 掌握常见程序威胁（特洛伊木马、后门、逻辑炸弹、缓冲区溢出、病毒）
- [ ] 理解系统威胁（蠕虫、端口扫描、DDoS）及其防护策略
- [ ] 掌握对称加密与非对称加密的原理与区别
- [ ] 理解数字签名、MAC、哈希函数的安全作用
- [ ] 掌握 PKI 体系与数字证书的工作原理
- [ ] 理解 SSL/TLS 握手过程与密钥协商机制
- [ ] 理解用户认证的基本原理
