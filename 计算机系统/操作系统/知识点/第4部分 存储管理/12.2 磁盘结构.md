# 12.2 磁盘结构

本节聚焦于**磁盘结构**，是[[第十二章 大容量存储设备]]中的独立知识节点。

从操作系统视角，设备提供一组可读写的逻辑块。控制器、固件和介质内部映射把逻辑块请求转换为具体位置，并可能重排请求、缓存写入或替换坏块。逻辑块大小、文件系统块大小、页大小和设备物理扇区大小是不同层次的单位，应在讨论对齐、碎片和 I/O 放大时分别说明。

```mermaid
flowchart LR
  A["文件偏移"] --> B["文件系统逻辑块"]
  B --> C["设备逻辑块地址（LBA）"]
  C --> D["控制器映射与缓存"]
  D --> E["介质页/扇区/擦除块"]
```

对机械磁盘，连续逻辑块通常有利于顺序传输，但现代设备的内部重映射使“逻辑连续”不总是等同于物理相邻。对 SSD，控制器的闪存转换层进一步增强了这一差异。
