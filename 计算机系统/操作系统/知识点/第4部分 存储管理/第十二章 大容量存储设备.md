# 第十二章 大容量存储设备

> [!abstract] 本章解决什么问题？
> 大容量存储为文件系统、虚拟内存和持久化服务提供底层容量。操作系统不仅要向设备发出读写请求，还要处理介质延迟、并发请求调度、格式化与坏块、交换空间、冗余阵列和断电后的数据一致性。本章讨论这些机制及其性能、可靠性取舍。

## 学习目标

- [ ] 能解释机械磁盘的寻道、旋转延迟和传输时间。
- [ ] 能根据请求队列比较常见磁盘调度算法的总移动距离与饥饿风险。
- [ ] 能区分直接连接存储、网络连接存储和存储区域网络。
- [ ] 能说明低级格式化、分区、逻辑格式化、引导块和坏块处理的关系。
- [ ] 能解释 RAID 如何利用冗余和并行性改善可靠性与性能。
- [ ] 能区分“写入完成”“持久化完成”和“稳定存储”的语义。
