# 06-输入与输出设备 (Chapter 6: Input and Output)

> 人机交互是计算机发挥作用的重要环节。我们通过**输入设备**将人类语言、动作和图像翻译成计算机能处理的 0 和 1；计算机则通过**输出设备**将机器语言翻译回我们能理解的文字、图像和声音。本章将带你认识形形色色的输入输出硬件。

## 🎯 学习目标 (Competencies)
阅读本章后，你应当能够：
1. 定义输入。
2. 描述键盘录入，包括键盘的类型和特性。
3. 讨论指点设备，包括游戏控制器和手写笔。
4. 描述扫描设备，包括光学扫描仪、RFID 阅读器和识别设备。
5. 讨论图像捕捉和音频输入设备。
6. 定义输出。
7. 讨论显示器的特性和类型，包括平板显示器和电子书。
8. 讨论打印机的特性和类型，包括喷墨打印机和云打印机。
9. 讨论音频和视频设备，包括便携式媒体播放器和移动数字电视 (Mobile DTV)。
10. 定义组合输入输出设备，包括多功能设备、网络电话、机器人和虚拟现实头显及手套。
11. 讨论人体工程学以及减少身体伤害的方法。

## ✍️ 什么是输入？ (What Is Input?)
**输入 (Input)** 是计算机使用的任何数据或指令。它可以直接来自我们，也可以来自其他源（如扫描仪、摄像机）。
- **输入设备 (Input devices)** 是将人类能理解的动作、文字、声音和图像，转化为系统单元能处理的**二进制机器语言 (0和1)** 的硬件。
- 常见的输入方式包括：打字（键盘）、指向（鼠标）、扫描（扫描仪）、拍照（数码相机）和说话（麦克风）。

### 1. 键盘录入 (Keyboard Entry)
键盘将数字、字母和特殊字符转换为**电信号**发送给系统单元。绝大多数键盘使用 **QWERTY** 键盘布局（取键盘第一排字母顺序的前六个字母命名）。
键盘主要分为四种类型：
- **传统键盘 (Traditional keyboards)**：常用于台式机。标准键盘为 101 键，包含功能键（如 F1 常显示帮助）、导航键和数字小键盘。有些按键是**切换键 (Toggle keys)**（如 Caps Lock），有些是**组合键 (Combination keys)**（如 Ctrl、Alt）。
- **笔记本键盘 (Notebook keyboards)**：常用于笔记本电脑或上网本。布局紧凑，通常没有独立数字小键盘，导航键位置也不固定。
- **虚拟键盘 (Virtual keyboards)**：常用于平板电脑或部分智能手机。没有物理按键，直接在触摸屏上显示并触控选择。
- **拇指键盘 (Thumb keyboards)**：常用于智能手机或其他小型便携设备。非常小，专为通过拇指快速输入文本和连接网络而设计。

### 2. 指点设备 (Pointing Devices)
指点设备通过接受点击和指向手势，并将其转换为机器可读的输入，提供直观的交互。
- **鼠标 (Mice)**：控制屏幕上的鼠标指针（通常为箭头形状）。最广泛使用的是**光电鼠标 (Optical mouse)**，通过感光检测移动。**无线鼠标 (Wireless mouse)** 通过无线电波或红外线与系统单元通信，释放了桌面空间。
- **轨迹球 (Trackballs)**、**触摸板 (Touch pads)** 和 **指点杆 (Pointing sticks)** 常用于替代鼠标，常见于笔记本电脑。
- **触摸屏 (Touch Screens)**：允许用户用手指或笔状物触摸屏幕来选择动作。**多点触控屏 (Multitouch screens)** 可同时感应多个手指，支持手势（如双指缩放旋转），广泛用于平板电脑和智能手机。
- **游戏控制器 (Game Controllers)**：提供输入以控制游戏。
  - 游戏杆 (Joysticks)、跳舞垫 (Dance pads)、游戏手柄 (Gamepads)、动作感应设备 (Motion-sensing devices，如微软 Xbox 360 的 Kinect)。
- **手写笔 (Stylus)**：一种笔状设备，常用于平板电脑或 PDA。配合**手写识别软件 (Handwriting recognition software)** 使用，将手写文字转换为系统单元可处理的文本。

### 3. 扫描设备 (Scanning Devices)
扫描设备以电子的方式读取文本和图像，并将其转换为机器可读的形式，以便处理、显示或打印。分为五类：
- **光学扫描仪 (Optical Scanners)**：将文本和图像转换为机器可读格式。常见类型有**平板式 (Flatbed)**、**馈纸式 (Document scanner)** 和**便携式 (Portable)**。
- **读卡器 (Card Readers)**：最常见的磁条阅读器 (Magnetic card reader) 可读取信用卡或身份证背后的磁条信息。
- **条码阅读器 (Bar Code Readers)**：通过光电管读取条形码。超市使用的通用产品代码 (UPC) 便是最典型的应用。智能手机上的 App 也可利用摄像头扫描商品条码进行比价。
- **RFID 阅读器 (RFID Readers)**：利用无线电频率识别技术，读取标签上的微芯片信息。常用于追踪宠物、管理库存或记录商品信息。
- **字符和标记识别设备 (Character and Mark Recognition)**：
  - **MICR (磁墨字符识别)**：银行用于自动读取支票下方的奇怪数字。
  - **OCR (光学字符识别)**：用光源读取印刷字符并转换为机器码。**手持式扫码枪 (Wand reader)** 是常见应用。
  - **OMR (光学标记识别)**：感测标记（如铅笔涂黑）的存在，常用于标准化多项选择题考试的阅卷。

### 4. 图像捕捉设备 (Image Capturing Devices)
与复印机复制不同，图像捕捉设备是**创造或捕捉**原始图像。
- **数码相机 (Digital Cameras)**：照片以数字形式记录在磁盘或相机内存中，大多支持视频录制。
- **网络摄像头 (Webcams)**：专门用于捕捉图像并发送到互联网的设备，常用于视频通话。许多笔记本电脑和智能手机内置了网络摄像头。

> ⚖️ **伦理思考 (Ethics)**：有人曾利用网络摄像头秘密记录他人活动（如宿舍室友）。在没有获得他人知情同意的情况下，拍摄并广播他人影像是**不道德**的行为。

### 5. 音频输入设备 (Audio-Input Devices)
将声音转换为系统单元可以处理的形式。最广泛使用的是**麦克风 (Microphone)**。
- **语音识别系统 (Voice Recognition Systems)**：利用麦克风、声卡和专用软件，让用户用语音命令操作电脑。GPS 导航、汽车音响的声控拨号都是典型应用。

## 📤 什么是输出？ (What Is Output?)
**输出 (Output)** 是系统单元处理后生成的、能被人类理解的数据或信息。**输出设备 (Output devices)** 将机器语言翻译成文字、图形、照片、音频或视频。

### 1. 显示器 (Monitors)
显示器是**最常用的输出设备**。它呈现视觉图像，被称为**软拷贝 (Soft copy)**。
- **分辨率 (Resolution)**：显示器上形成的图像由称为**像素 (Pixels)** 的点阵组成。像素越多，图像越清晰。如 UXGA 标准为 1,600 × 1,200 像素。
- **点距 (Dot pitch)**：像素之间的距离。点距越小，图像越清晰。
- **对比度 (Contrast ratio)**：显示最亮白色和最暗黑色的对比程度。对比度越高，显示器越好。
- **尺寸 (Size)**：显示器对角线长度，常见的有 15、17、19、21 英寸。
- **宽高比 (Aspect ratio)**：显示器宽度除以高度。标准显示器多为 4:3，宽屏显示器为 16:9。

**显示器类型**：
- **平板显示器 (Flat-Panel Monitors)**：最广泛使用的类型。轻薄、便携且功耗低。
  - **LCD (液晶显示器)**：通过背光发光，市场上最常见的是 **TFT-LC (薄膜晶体管液晶)**。
  - **AMOLED (有源矩阵有机发光二极管)**：相比 LCD，功耗更低、电池寿命更长、机身更薄。
- **电子书阅读器 (E-book readers)**：专用移动设备。屏幕仅 6 英寸左右，只支持黑白输出，使用 **E-ink (电子墨水)** 技术（反射自然光，看起来像普通纸张），如 Amazon Kindle、Nook。
- **其他显示器 (Other Monitors)**：数字化交互式白板 (Digital interactive whiteboards) 用于演示；高清晰度电视 (HDTV)；**CRT (阴极射线管)** 显示器已被淘汰，但含有约 4 磅铅，必须通过环保认证的回收计划处理。

### 2. 打印机 (Printers)
打印机将系统单元处理的信息翻译到纸张上，被称为**硬拷贝 (Hard copy)**。
**常见特性**：
- **分辨率 (Resolution)**：以 **DPI (每英寸点数)** 表示。DPI 越高，图像越清晰。
- **色彩 (Color capability)**：可以打印纯黑、灰度（大量灰度阴影）或彩色。
- **速度 (Speed)**：以 PPM (每分钟页数) 表示。
- **内存 (Memory)**：打印机内存越大，处理大型文档越快。
- **双面打印 (Duplex printing)**：自动在纸张双面打印，减少纸张浪费，保护环境。

**常见类型**：
- **喷墨打印机 (Inkjet Printers)**：将墨水高速喷射到纸面上。色彩还原好，能打印高质量照片。价格便宜，但墨盒昂贵。
- **激光打印机 (Laser Printers)**：利用激光光束打印出类似复印机的效果。打印速度快，打印质量极佳，但价格较昂贵。分为**个人激光打印机 (Personal laser printers)** 和**共享激光打印机 (Shared laser printers)**。
- **其他打印机**：**云打印机 (Cloud printers)** 可让用户随时随地连接网络打印（如 Google Cloud Print）；**热敏打印机 (Thermal printers)** 用于 ATM 机和加油站收据；**绘图仪 (Plotters)** 用于打印建筑蓝图和工程图纸。

> 🎬 **Making IT Work for You：电子书 (E-Books)**
> 电子书相比传统纸质书的巨大优势在于：**便利运输**（一本设备藏千本书）、**快速搜索**（搜索关键词直达页面）、**数字书签**、**高亮和添加笔记**（可随时移除）、**低价便捷购买**（不到1分钟下载完毕）。你可以在专用电子书阅读器、智能手机/平板 App、笔记本/台式机上阅读。

### 3. 音频与视频设备 (Audio and Video Devices)
- **音频输出设备 (Audio-output devices)**：将音频信息翻译成人类能理解的声音。最广泛使用的是**扬声器 (Speakers)** 和**耳机 (Headsets)**，连接在系统单元的音频接口上，可播放音乐、语音合成等。
- **便携式媒体播放器 (Portable media players)**：存储和播放数字媒体的电子设备，如 Apple iPod。最新的应用是**移动数字电视 (Mobile DTV)**，允许电视直接向智能手机播放节目。

## 🔄 组合输入与输出设备 (Combination Input and Output Devices)
许多设备将输入和输出能力结合在一个单元中，以节省空间或实现特殊应用。
- **多功能设备 (Multifunctional Devices, MFD)**：通常将扫描仪、打印机、传真机和复印机的能力集成一体，价格和占用的空间都比单独购买要少很多。
- **网络电话 (Internet telephones)**：即 **VoIP (基于 IP 的语音传输)**。如 **Skype**，允许用户通过互联网进行免费语音和视频通话。弊端是通话双方都需有电脑或 App，且通话质量依赖网络速度。
- **机器人 (Robots)**：计算机控制的机器，模仿生物（如人类）的肢体动作。有四大类：
  - **感知机器人 (Perception system robots)**：模仿视觉、触觉等感官。
  - **工业机器人 (Industrial robots)**：用于焊接、喷漆、组装。
  - **移动机器人 (Mobile robots)**：作为运输工具，用于排爆或侦查。
  - **家用机器人 (Household robots)**：用于吸尘、割草或提供家庭娱乐。

- **虚拟现实 (Virtual Reality, VR)**：通过电脑创建的 3D 人工或模拟体验。需要特殊硬件，如**头戴式显示器 (Headgear)** 和**数据手套 (Gloves)** 来沉浸其中。
> ⚖️ **伦理思考 (Ethics)**：VR 可能让参与者沉浸在不道德的情景或暴力游戏中。有人担忧长时间沉浸在这些环境，会使人脱敏，无法应对现实世界的规则。

## 🧘 人体工程学 (Ergonomics)
**人体工程学**是对与人们使用产品相关的人类因素的研究。它关注的是**让工作适应人**，而不是让人去适应工作，具体体现在输入输出设备的设计上。
长期使用电脑不当会导致：
- **眼疲劳和头痛 (Eyestrain and headache)**：建议每小时休息 15 分钟。
- **背部和颈部疼痛 (Back and neck pain)**：使用可调节的椅子、显示器与视线平齐、使用脚踏板。
- **重复性劳损 (Repetitive strain injury, RSI)**：由于快速、重复的工作导致手腕或手臂受损。最典型的类型是**腕管综合征 (Carpal tunnel syndrome)**。采用符合人体工程学的键盘，经常按摩手腕、短暂休息有助于预防。

## 🧑‍💻 IT 职业：技术文档撰写员 (Careers in IT: Technical Writer)
**技术文档撰写员**负责编写操作手册、技术报告和其他科学/技术文档。他们大多就职于软件公司、政府机构或研究机构，将专业技术信息翻译成**大众易于理解的说明或总结**。
- **教育/技能要求**：通常需要通信、新闻学或英语专业的**副学士或大学学位**，并对特定的**技术领域有专长或了解**。有很强写作能力的人常从科学领域转行过来。
- **职业发展**：可向 API 文档、开发者体验、信息架构、内容设计或技术传播管理发展。薪酬受地区、行业与专业深度影响，应查询最新地域统计。

## ✅ 关键术语速查 (Key Terms Check)
- **DPI (每英寸点数)**：衡量打印机输出清晰度的单位，越高越清晰。
- **LCD / AMOLED**：两种主要平板显示技术。LCD 背光发光，AMOLED 像素自发光且更省电。
- **E-ink (电子墨水)**：电子书阅读器使用的技术，靠反射自然光显示，无背光，类似纸质书。
- **VoIP (IP语音传输)**：利用互联网而不是传统电话线来进行语音通话的技术（如 Skype）。
- **OCR / OMR / MICR**：光学字符识别、光学标记识别、磁墨字符识别。
- **RSI (重复性劳损)**：长时间重复性动作导致的手部或手臂损伤，最著名的类型是腕管综合征。

## 🔁 人机交互闭环

```mermaid
flowchart LR
  U[用户目标] --> IN[输入<br/>触摸、键盘、语音、传感器]
  IN --> SYS[系统处理]
  SYS --> OUT[输出<br/>视觉、听觉、触觉]
  OUT --> P[用户感知与判断]
  P --> U
```

好的交互设计不仅要求设备“能输入、能输出”，还要让反馈及时、状态可见、错误可恢复。例如点击“提交”后立即显示进度，可避免用户因不确定而重复提交。

## 📐 分辨率不是质量的全部

| 对象 | 常见指标 | 还需考虑 |
|---|---|---|
| 显示器 | 像素分辨率、刷新率 | 尺寸、像素密度、色域、亮度、响应时间 |
| 相机 | 像素数量 | 传感器尺寸、镜头、动态范围、算法 |
| 打印机 | DPI | 墨水、纸张、色彩管理、打印速度 |
| 麦克风 | 采样率、位深 | 信噪比、指向性、环境声学 |

> [!example] 无障碍案例
> 只用颜色区分“成功/失败”会让部分色觉障碍用户难以识别。应同时使用文字、图标或形状；交互还应支持键盘、屏幕阅读器、字幕和合理的触控目标。

> [!warning] 人体工程学
> 固定的“每小时休息多少分钟”不是适合所有人的医学处方。更稳妥的原则是频繁改变姿势、进行短暂休息、减少持续重复动作，并在不适持续时寻求专业建议。
