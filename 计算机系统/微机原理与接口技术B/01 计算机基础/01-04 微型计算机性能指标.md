## 1.1.4 微型计算机性能的评估指标

一台微型计算机性能的优劣是由其系统结构、指令系统、硬件组成、外部设备、软件配备是否齐全等因素决定的。只有综合各项指标才能正确评价与衡量计算机性能的高低。下面介绍几项主要评估指标。

### 1. CPU 字长
计算机的字长决定了计算机内部一次可以处理的二进制代码的位数。它决定着计算机通用寄存器、加法器、数据总线等部件的位数，因此直接影响硬件成本。一般地，一台计算机的字长由下列一个或两个条件所决定：① ALU 一次能处理的二进制编码位数；② 数据总线的宽度；③ 内存储器的字长。

字长越长，一个字能表示的数据精度越高。在完成同样精度的运算时，字长较长的计算机比字长短的计算机速度快。为了兼顾精度与硬件成本，有些计算机允许采用变字长运算。

一般情况下，CPU 内部数据总线宽度和计算机对外的数据总线宽度是一致的。早期有些 CPU 为了配合外设的总线宽度，设计内部字长和外部数据总线宽度不一致。如 Intel 8088 微处理器就是内部总线宽度为 $16$，而外部数据总线宽度是 $8$ 位的芯片，称为“准 $16$ 位”芯片。而后期的 CPU，则提供比内部数据宽度更高的总线宽度来访问内存，以提高访问速度。如 Pentium 处理器的字长为 $32$ 位，但其访问内存的外部数据总线宽度为 $64$ 位。

### 2. 内存储器与高速缓存
内存储器容量是衡量存储二进制信息量的重要指标。通常用字节表示内存储器的容量，如 $64 \text{ MB}$。内存储器容量范围大小根据其用途、成本及价格等因素选择。

内存储器的速度可用访问延迟、周期时间和带宽等指标衡量。存取周期表示存储器连续启动两次独立访问所需的最小时间间隔，不能与单次访问延迟或持续传输带宽混为一谈；不同 SRAM、DRAM 及其工作方式的指标差异很大，应结合器件和接口时序分析。

CPU 与主存之间存在明显的速度差距，高速缓存（Cache）利用程序访问的时间局部性和空间局部性降低平均访存延迟。缓存容量只是影响性能的因素之一，命中率、延迟、相联度、替换策略、层级组织与一致性开销同样重要；容量增大也可能带来更长访问延迟和更高面积、功耗成本。

### 3. CPU 指令执行时间
指令执行时间的长短反映 CPU 运算速度的快慢。因为执行不同的指令所需的时间不同，就产生了如何评估计算速度的问题。常用的方法有如下两种。

1. 指令执行的平均速度值

根据不同类型指令在计算过程中出现的频繁程度乘上不同系数，可求得统计平均值，这里的运算速度指平均速度 MIPS（Millions of Instruction Per Second），以百万条指令/秒为单位；其全名为 Drytone V1.1 MIPS，即运行一个名为 Drytone 的测试程序，测试性能规范到 VAX1.1 计算机，如 486DX 性能相当于每秒执行 2700 万条指令的 VAX1.1 计算机，称为 27VAX MIPS。

2. CPU 的主频

主频是 CPU 的时钟频率，很大程度上决定了计算机的运算速度，是 CPU 的标称频率。如 80486 时钟频率为 $33 \sim 66 \text{ MHz}$，Pentium 则为 $60 \sim 133 \text{ MHz}$，Pentium MMX CPU 时钟频率为 $230 \text{ MHz}$ 以上。

在 80486 CPU 出现后，由于 CPU 频率不断提高，外接设备的速度却受到工艺的限制，不能承受更高的频率，因此限制了 CPU 频率的进一步提高，出现了倍频技术。该技术能够使 CPU 内部工作频率变为外部频率的倍数，从而可通过提升倍频而达到提升主频的目的。当主频升至 $3 \text{ GHz}$ 后，功耗和散热等因素使得频率不适宜进一步提升，为了优化处理器的工作速度，采用的解决办法是多核设计。

### 4. 系统总线的传输速率
系统总线的有效传输速率会直接影响 I/O 性能，其上限与每次传输的数据宽度、传输频率、编码开销、协议开销和通道数量有关。并行 PCI 的理论峰值可用“总线频率 × 总线宽度”估算，例如 33 MHz、32 位 PCI 的峰值约为 $133\ \text{MB/s}$。PCI Express（PCIe）采用点到点串行链路，带宽必须同时注明**代际、通道数和方向**；不能仅凭“PCIe ×16”给出一个跨代通用数值。实际吞吐量通常低于理论峰值。

> [!tip] 带宽计算口径
> 比较总线性能时，应先确认数值表示单向还是双向、原始传输率还是有效载荷速率，以及使用 MB/s（$10^6$ B/s）还是 MiB/s（$2^{20}$ B/s）。

### 5. 优化的内部结构
CPU 体系的发展过程中采用了许多新技术，优化了其软件、硬件结构，提升了 CPU 的整体性能。在这些新技术中，指令流水线结构提升了系统的并行处理能力，从而加快了指令执行时间，而高速缓存（Cache）结构加快了 CPU 访问数据和指令的速度，这两项技术的应用可以说是优化 CPU 性能的关键技术。除此之外，高端 CPU 往往在其设计实现中进行了多方面、多层次的优化，显著提升了其性能。优化技术包括：超流水线和超标量技术、乱序执行技术、分支预测执行技术等。

超流水线是指某些 CPU 内部的流水线超过通常的 $5 \sim 6$ 步，如 Pentium CPU 的流水线为 $14$ 步，流水线设计的步数越多，其完成一条指令的速度越快。超标量是指在 CPU 中有 $1$ 条以上的流水线，并且每个时钟周期内可以完成一条以上的指令。

乱序执行和分支预测执行技术是高端 CPU 进行动态处理的核心技术，动态处理采用多路分流预测算法，在取指同时寻找将要执行的指令，该技术可加速向处理器传送任务，通过提前判读并执行有可能需要的指令的方式提高执行速度，可使高端 CPU 的处理能力得到充分的发挥，从而提升系统性能。

### 6. I/O 设备配备情况
在现代计算机系统中，I/O 设备配备的性能对整体系统的综合表现具有重大的影响。在微型计算机系统中，外存储器容量、打印机型号、显示屏幕的分辨率，都是 I/O 设备选用中要考虑的问题。

### 7. 软件配备情况
软件是计算机系统不可缺少的重要组成部分，配备是否齐全直接关系到计算机性能的好坏、效率的高低。这里主要指操作系统、高级语言和汇编语言的种类、应用软件等。
